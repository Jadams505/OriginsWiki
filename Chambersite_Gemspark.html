Chambersite Gemspark

[statblock Chambersite_Gemspark]

The Chambersite Gemspark is a type of gemspark block crafted from Glass and Chambersite. Like the others, it glows in the dark, which is additionally toggleable using wires. If a waterfall passes through a Chambersite Gemspark, it will color it and cause it to glow while it is active.

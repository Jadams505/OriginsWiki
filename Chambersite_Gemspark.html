Chambersite Gemspark

[statblock Chambersite_Gemspark]

The Chambersite Gemspark is a type of [link gemspark | | https://terraria.wiki.gg/wiki/Gemspark_Blocks] [link block | | Blocks.html] crafted from [link Glass | | https://terraria.wiki.gg/wiki/Glass] and [link Chambersite]. Like the others, it glows in the dark, which is additionally toggleable using [link wires | | https://terraria.wiki.gg/wiki/Wire]. If a [link waterfall | | https://terraria.wiki.gg/wiki/Liquidfalls] passes through a Chambersite Gemspark, it will color it and cause it to glow while it is active.

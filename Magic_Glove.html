Magic Glove

[statblock Magic_Glove]

The Magic Glove is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link accessory | | Accessories.html] combining a [link Mana Cloak | | https://terraria.wiki.gg/wiki/Mana_Cloak] and a [link Gun Glove]. Swinging while this is equipped will create a random magic projectile from the player's swinging-hand.

Magic Glove

[statblock Magic_Glove]

The Magic Glove is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link accessory | | Accessories.html] combining a [link Mana Cloak | | https://terraria.wiki.gg/wiki/Mana_Cloak] and a [link Gun Glove]. Swinging while this is equipped will create a random magic projectile from the player's swinging-hand. It also provides an 8% reduction in [link mana | | https://terraria.wiki.gg/wiki/Mana] consumption and allows the player to automatically consume mana-restoring [link potions | | https://terraria.wiki.gg/wiki/Potions] in the their [link inventory | | https://terraria.wiki.gg/wiki/Inventory].

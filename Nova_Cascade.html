Nova Cascade

[statblock Nova_Cascade]

The Nova Cascade is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode], post-[link Lunatic Cultist | | https://terraria.wiki.gg/wiki/Lunatic_Cultist] [link handcannon | | Handcannons.html] crafted with [link Nova Fragments]. It behaves like other handcannons except that its slugs explode everytime they pierce an enemy for up to a maximum of 7 enemies.

The best [link modifier | | Modifiers.html] is Guileless.

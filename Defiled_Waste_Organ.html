Defiled Waste Organ

[statblock Defiled_Waste_Organ]

The Defiled Waste Organ is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link crafting station | | Crafting_Stations.html] used to craft [link furniture] based on the [link Defiled | | Defiled_Wastelands.html]-themed [link Cancer Block]. The Defiled Waste Organ can be purchased from the [link Steampunker | | https://terraria.wiki.gg/wiki/Steampunker] for [price 10|0|0] but only appears for sale in worlds that are generated with the Riven.

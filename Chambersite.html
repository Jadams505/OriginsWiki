Chambersite

[statblock Chambersite]

Chambersite is a [link gem | | https://terraria.wiki.gg/wiki/Gem] found exclusively in the [link Underground | | https://terraria.wiki.gg/wiki/Underground] [link evil biomes | | https://terraria.wiki.gg/wiki/Evil_biomes]. When a world evil [link spreads | | https://terraria.wiki.gg/wiki/Biome_spread] naturally or through [link Clentaminator | | https://terraria.wiki.gg/wiki/Clentaminator] and [link solution | | https://terraria.wiki.gg/wiki/Solutions], any adjacent gem stone blocks will be converted to a Chambersite equivalent. Chambersite gem stones can appear in the [link Stone | | https://terraria.wiki.gg/wiki/Stone_Block] archetype of the present evil, except for the [link Ashen Factory]. Chambersite gem stones cannot be converted by the [link the Hallow | | https://terraria.wiki.gg/wiki/The_Hallow], and will instead revert to their pure form. Furthermore, all of the archetypes and their walls are craftable at a [link Heavy Work Bench | | https://terraria.wiki.gg/wiki/Heavy_Work_Bench] in a [link Graveyard | | https://terraria.wiki.gg/wiki/Graveyard] biome. Chambersites do not have their own [link Gem Tree | | https://terraria.wiki.gg/wiki/Trees#Gem_Trees], nor can they be obtained by smashing [link Geodes | | https://terraria.wiki.gg/wiki/Geode].

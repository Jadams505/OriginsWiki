Mildewy Diving Helmet

[statblock Mildewy_Diving_Helmet]

The Mildewy Diving Helmet is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link accessory | | Accessories.html] that is dropped by the [link Lost Diver] in [link Master Mode | | https://terraria.wiki.gg/wiki/Master_Mode]. While this is equipped, will be able to transform into a mildew carrion for 25 seconds after double-tapping the Down button, which provides the player with 50% more [link defense | | https://terraria.wiki.gg/wiki/Defense] and [link damage | | https://terraria.wiki.gg/wiki/Damage]. While transformed, the player will not be able to [link fly | | https://terraria.wiki.gg/wiki/Flight] or benefit from any movement accessories other than [link Super Turbo Reel] and its upgrades, which will increase the pulling strength of the carrion-form's claws. Once the duration of the transformation has elapsed, the player will go on a 25-second cooldown before the ability can be used again.

Edengrass

[statblock Edengrass]

Edengrass is a [link block | | Blocks.html] type found in [link the Dawn]. It is the Dawn-equivalent to [link Grass | | https://terraria.wiki.gg/wiki/Grasses], and cannot be converted to other variants with [link any thrown powder | | https://terraria.wiki.gg/wiki/Thrown_Powder]. Edengrass can be placed and spread using the [link Seeds of Eden]. Given enough time, [link foliage | | Plants.html] will start to grow on top of Edengrass.

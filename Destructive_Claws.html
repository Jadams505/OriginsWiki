Destructive Claws

[statblock Destructive_Claws]

Destructive Claws are a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link accessory | | Accessories.html] crafted using Feral Claws and the [link Bomb Handling Device | | Pneumatic_Bomb_Thrower.html]. It increases the velocity of thrown [link explosives | | Demolitionist_Weapons.html] by 25% and their attack speed by 10%. Additionally, it enables auto-use for all explosive weapons.

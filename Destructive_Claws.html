Destructive Claws

[statblock Destructive_Claws]

Destructive Claws are a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] crafted using [link Feral Claws | | https://terraria.wiki.gg/wiki/Feral_Claws] and the [link Bomb Handling Device]. It increases the velocity of [link thrown | | https://terraria.wiki.gg/wiki/Throwing_weapons] [link explosives | | Demolitionist_Weapons.html] by 25% and their attack speed by 10%. Additionally, it enables [link auto-use | | https://terraria.wiki.gg/wiki/Autoswing] for all explosive weapons.

Peat Moss

"The Demolitionist might find this interesting..."

[statblock Peat_Moss]

Peat Moss is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link material] and [link block | | Blocks.html] type found in the [link Brine Pool]. It envelops the outermost parts of each cell within the Brine Pool, but it can be grown on any [link mud | | https://terraria.wiki.gg/wiki/Mud_Block] block given it is submerged. It can additionally be dropped by many enemies in the [link Underground Jungle | | https://terraria.wiki.gg/wiki/Underground_Jungle]. Peat Moss can be sold to the [link Demolitionist | | https://terraria.wiki.gg/wiki/Demolitionist] NPC to unlock some [link explosive | | Demolitionist_Weapons.html] items that which he can sell.

Peat Moss

{statblock { header:Peat Moss, items:[ {image: Images/Peat_Moss} ] }, { header:Statistics, items:[ { label:Max Stack, value:999 }, { label:Tooltip, value:"The Demolitionist might find this interesting..." }, { label:Rarity, value:White }, { label:Sell, value:3 Silver }, { label:Research, value:99 required } ] } statblock}

Peat Moss is a pre-Hardmode material that is used to craft some early game [link Demolitionist Weapons]. It is harvested from [link Peat] blocks found in the Jungle biome, especially on the surface of the [link Brine Pool]. It can be grown on
any Mud tile next to a water source whilst in a Jungle biome. It is also crafted into [link Explosive Powder] for advanced crafting.

Ashen Armor

[statblock Ashen_Armor]

Ashen armor is an [link armor] set consisting of the Ashen Helmet, Ashen Breastplate and Ashen Greaves. In order to craft the full set the player will need 60 [link Sanguinite Bars | $default | Sanguinite_Bar.html] (180 [link Sanguinite Ore]) and 45 [link NE8].

A full set of Ashen armor provides the following benefits:

- +19 [link defense | $default | https://terraria.wiki.gg/wiki/Defense]
- +8% damage
- +8% attack speed
- +8% [link critical strike | $default | https://terraria.wiki.gg/wiki/Critical_hit] chance
Set bonus
- +15% knockback and knockback resistance
- -15% [link self-damage | $default | Self_Damage.html]

Ashen Armor

[statblock Ashen_Armor]

Ashen armor is an [link armor] set consisting of the Ashen Helmet, Ashen Breastplate and Ashen Greaves. In order to craft the full set the player will need 60 [link Sanguinite Bars | | Sanguinite_Bar.html] (180 [link Sanguinite Ore]) and 45 [link NE-8 | | NE_8.html].

A full set of Ashen armor provides the following benefits:

- +19 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +8% damage
- +8% [link attack speed | | https://terraria.wiki.gg/wiki/Attack_speed]
- +8% [link critical strike | | https://terraria.wiki.gg/wiki/Critical_hit] chance
Set bonus
- +15% [link knockback | | https://terraria.wiki.gg/wiki/Knockback] and knockback resistance
- -15% [link self-damage | | Self_Damage.html]

Purification Potion

[statblock Purification_Potion]

The Purification Potion is a [link buff potion | | Buff_Potions.html] that can be crafted using [link Ectoplasm | | https://terraria.wiki.gg/wiki/Ectoplasm], [link Moonglow | | https://terraria.wiki.gg/wiki/Moonglow] and a [link Mojo Flask]. When consumed, it provides the Willed [link buff | | Buffs.html], which provides immunity to and removes all current progress of any kind of [link Assimilation]. This effect will last for 6 minutes, but can be canceled anytime by right-clicking the icon.

Left Click

[statblock Left_Click]

The Left Click is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link harpoon gun | | Harpoon_Guns.html] that has a 1/99 (1.01%) chance of being received by the [link Defiled Loot Box] dropped by enemies during the [link Defiled Invasion] [link event | | Events.html]. When using regular [link harpoons | | Harpoon.html], a random mouse-pointer projectile has a chance of being fired by this weapon. The loading icon inflicts [link Rasterized] for 2 seconds, the select icon inflicts [link Ichor | | https://terraria.wiki.gg/wiki/Ichor_(debuff)] debuff, the drag icon pulls enemies toward the player, and the x-icon guarantees a [link critical hit | | https://terraria.wiki.gg/wiki/Critical_hit].

Its best [link modifier | | Modifiers.html] is Godly or Demonic.

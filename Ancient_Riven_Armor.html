Ancient Riven Armor

[statblock Ancient_Riven_Armor]

Ancient Riven armor is an [link armor] set that reproduces the pre-release design, and is otherwise identical in all ways to the current [link Riven armor] set. Each piece is interchangeable with the equivalent Defiled armor piece, while still granting the same set bonus.

Each piece has a 1/525 (0.19%) chance of being dropped by most [link Riven | | Riven_Hive.html] enemies.

Magic Pain Glove

[statblock Magic_Pain_Glove]

The Magic Pain Glove is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link accessory | | Accessories.html] combining [link Feral Claws | | https://terraria.wiki.gg/wiki/Feral_Claws] and a [link Magic Glove]. The Glove increases melee [link attack speed | | https://terraria.wiki.gg/wiki/Attack_speed] by 12% and generates a random magic projectile while the player's hand is being swung. It also grants [link auto-use | | https://terraria.wiki.gg/wiki/Autoswing] to all [link melee weapons] and [link whips]. Furthermore, it also provides an 8% reduction in [link mana | | https://terraria.wiki.gg/wiki/Mana] consumption and allows the player to automatically consume mana-restoring [link potions | | https://terraria.wiki.gg/wiki/Potions] in the their [link inventory | | https://terraria.wiki.gg/wiki/Inventory].

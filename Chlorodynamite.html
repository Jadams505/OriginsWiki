Chlorodynamite

[statblock Chlorodynamite]

Chlorodynamite is a post-[link Golem | | https://terraria.wiki.gg/wiki/Golem] [link explosive | | Demolitionist_Weapons.html] that can be crafted using [link Chlorophyte Ore | | https://terraria.wiki.gg/wiki/Crimtane_Ore] or [link Chlorophyte Bars | | https://terraria.wiki.gg/wiki/Crimtane_Bar] and [link Dynamite | | https://terraria.wiki.gg/wiki/Dynamite]. Before detonation, vines will reach out from the projectile and attempt to pull in any nearby enemies towards it. Chlorodynamite then explodes into a flash of light that inflicts [link Confused | | https://terraria.wiki.gg/wiki/Confused] for 3.67 seconds and [link Slow | | https://terraria.wiki.gg/wiki/Slow] for 5 seconds, similarly to the [link Flashbang].

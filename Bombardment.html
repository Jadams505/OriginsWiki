Bombardment

[statblock Bombardment]

The Bombardment is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link mine | | Mines.html] [link launcher | | Launchers.html] that can be crafted using [link Defiled Bars | | Defiled_Bar.html] and [link Undead Chunks | | Undead_Chunk.html]. The Bombardment converts its [link ammunition] into hovering explosives, that remain in place for 7 seconds before exploding or until coming to contact with an enemy. It also has a 50% chance [link not to consume ammo | | https://terraria.wiki.gg/wiki/Ammo_conservation].

Its best [link modifier | | Modifiers.html] is Guileless.

Felnum Ore

[statblock Felnum_Ore]

Felnum Ore is a type of [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link ore | | Ores.html] [link block | | Blocks.html] found in [link Floating Islands | | https://terraria.wiki.gg/wiki/Floating_Island] once [link Skeletron | | https://terraria.wiki.gg/wiki/Skeletron] has been defeated. Felnum Ore requires at least 65% [link pickaxe power | | https://terraria.wiki.gg/wiki/Pickaxe_power] to mine.

Felnum Ore generates electrical sparks when placed that light up the area.

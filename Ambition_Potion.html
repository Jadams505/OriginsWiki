Ambition Potion

[statblock Ambition_Potion]

The Ambition Potion is a [link buff potion | | Buff_Potions.html] that can be crafted using [link Bottled Water | | https://terraria.wiki.gg/wiki/Bottled_Water], a [link Polyeel] and any [link evil herb | | Recipes.html]. When consumed, it provides the Ambition [link buff | | Buffs.html], which increases damage, critical strike chances, and weapon speeds by 2.5% whilst also providing 12.5% [link stat share]. This effect will last for 4 minutes, but can be canceled anytime by right-clicking the icon.

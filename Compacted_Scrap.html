Compacted Scrap

[statblock Compacted_Scrap]

Compacted Scrap is a [link block | | Blocks.html] type found in the [link Ashen Factory]. It is the Ashen-equivalent to [link Stone | | https://terraria.wiki.gg/wiki/Stone_Block], and cannot be converted to other variants with [link any thrown powder | | https://terraria.wiki.gg/wiki/Thrown_Powder] unlike the other five types. Compacted Scrap is additionally not capable of [link spreading | | https://terraria.wiki.gg/wiki/Biome_spread] to adjacent tiles. Thus, Compacted Scrap does not behave like a conventional [link soil | | Soils.html] despite classifying as one. The only way to remove Compacted Scrap is by mining it with a [link pickaxe | | Pickaxes.html] or [link drill | | Drills.html], or by destroying it with an [link explosive | | Demolitionist_Weapons.html].

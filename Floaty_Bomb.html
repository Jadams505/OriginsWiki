Floaty Bomb

[statblock Floaty_Bomb]

The Floaty Bomb is an [link explosive | | Demolitionist_Weapons.html] that can be crafted using [link Bombs | | https://terraria.wiki.gg/wiki/Bomb], [link Bones | | https://terraria.wiki.gg/wiki/Bone], [link Clouds | | https://terraria.wiki.gg/wiki/Cloud] and [link Feathers | | https://terraria.wiki.gg/wiki/Feather]. The Floaty Bomb has no difference from regular bombs other than an increased [link velocity | | https://terraria.wiki.gg/wiki/Velocity] and 10% reduction the force [link gravity | | https://terraria.wiki.gg/wiki/Gravity] applies to it.

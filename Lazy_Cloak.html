Lazy Cloak

[statblock Lazy_Cloak]

The Lazy Cloak is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link accessory | | Accessories.html] that is obtained from the [link Treasure Bag | | https://terraria.wiki.gg/wiki/Treasure_Bag] dropped by the [link Severus] in [link Master Mode | | https://terraria.wiki.gg/wiki/Master_Mode]. While equipped, the Lazy Cloak will go after any enemies [link tagged | | https://terraria.wiki.gg/wiki/Whips] by the player and inflict the [link Ichor | | https://terraria.wiki.gg/wiki/Ichor_(debuff)] debuff. There is a unique visual effect on the player that can be turned off while equipped on the player. Additionally, the cloak will not be visibly worn by the player while it is chasing an enemy.

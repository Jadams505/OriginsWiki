Eruption

[statblock Eruption]

The Eruption is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link launcher | | Launchers.html] crafted using [link Hellstone Bars | | https://terraria.wiki.gg/wiki/Hellstone_Bar]. It converts its [link ammunition] into miniature geysers, which remain in place until stepped on by an enemy, releasing hot magma upwards into a narrow row and inflicting the [link On Fire! | | https://terraria.wiki.gg/wiki/On_Fire!] debuff.

Its best [link modifier | | Modifiers.html] is Guileless.

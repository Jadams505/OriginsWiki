Self-Destruct

[statblock Self_Destruct]

Self-Destruct is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link explosive weapon | | Demolitionist_Weapons.html] crafted using mech components dropped by the [link mechanical bosses | | https://terraria.wiki.gg/wiki/Mechanical_bosses]. When used, the player will charge up for 1.3 seconds before exploding themselves, dealing impressive damage in a 22-tile radius. All [link self-damage | | Self_Damage.html] reduction greater than or equal to 80% will be ignored for a maximum self-inflicted damage of 125 and a minimum of 26. Additionally, a glowing red button will appear on the player's chest similar to [link Eyndum Cores] while this is selected in the hotbar.

The best [link modifier | | Modifiers.html] is Guileless.

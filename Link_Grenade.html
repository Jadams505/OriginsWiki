Link Grenade

[statblock Link_Grenade]

The Link Grenade is an [link explosive | | Demolitionist_Weapons.html] that can be crafted using [link Grenades | | https://terraria.wiki.gg/wiki/Grenade] and [link Sanguinite Ore]. They are also sold by the [link Demolitionist | | https://terraria.wiki.gg/wiki/Demolitionist] [link NPC | | NPCs.html] for [price 3|0] after selling him 85 [link Peat Moss]. Link Grenades have a 20 second [link fuse time] and will only detonate if another explosive detonates nearby or after the fuse time has elapsed.

Residue Block

[statblock Residue_Block]

The Residue Block is a [link block | | Blocks.html] type crafted from 2 [link Spug Flesh] at a [link Meat Grinder | | https://terraria.wiki.gg/wiki/Meat_Grinder]. They are used to craft Residue [link furniture] and [link walls] at a [link Flesh Shredder]. Residue Blocks will not spread the [link Riven | | Riven_Hive.html].

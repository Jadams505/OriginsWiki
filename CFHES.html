C.F.H.E.S.

[statblock CFHES]

The C.F.H.E.S. is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] crafted using an [link ACME Crate] and [link Danger Barrel]. When equipped, all [link explosives | | Demolitionist_Weapons.html] inflict [link On Fire! | | https://terraria.wiki.gg/wiki/On_Fire!], have a 33% reduced [link fuse time | | Fuse_Time.html], an 80% increase to their blast radius and can detect enemies within proximity of their blast.

The C.F.H.E.S. is one of the strongest and latest-game Demolitionist accessories, which makes it a valuable boost to the player's combat effectiveness if obtained early on.

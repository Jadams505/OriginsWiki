Priority Mail

[statblock Priority_Mail]

Priority Mail is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] crafted using a [link Book | | https://terraria.wiki.gg/wiki/Book], [link Fallen Stars | | https://terraria.wiki.gg/wiki/Fallen_Star], [link Paper Airplanes | | https://terraria.wiki.gg/wiki/Paper_Airplanes] and an [link Asylum Whistle]. While equipped, enemies recently struck by an ally or [link minion | | https://terraria.wiki.gg/wiki/Summon_weapons#Minions] recieve 18% more damage from the player, and [link minion damage | | https://terraria.wiki.gg/wiki/Damage#Summon] is increased by 10%. Additionally, the player is able to [link tag | | https://terraria.wiki.gg/wiki/Whips] two different enemies at once.

Priority Mail is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] crafted using a [link Book | | https://terraria.wiki.gg/wiki/Book], [link Fallen Stars | | https://terraria.wiki.gg/wiki/Fallen_Star], [link Paper Airplanes | | https://terraria.wiki.gg/wiki/Paper_Airplanes] and an [link Asylum Whistle]. While equipped, enemies recently struck by a teammate recieve 18% more damage from the player. This accessory is only effective in [link multiplayer | | https://terraria.wiki.gg/wiki/Multiplayer] worlds.

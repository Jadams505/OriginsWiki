Necromancer Armor

[statblock Necromancer_Armor]

Necromancer armor is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode], post-[link Mech | | https://terraria.wiki.gg/wiki/Mechanical_bosses] [link armor] set consisting of the Necromancer Crown, Necromancer Breastplate and Necromancer Greaves. In order to craft the full set the player will need 54 [link Chlorophyte Bars | | https://terraria.wiki.gg/wiki/Chlorophyte_Bar], 3 [link Dark Shards | | https://terraria.wiki.gg/wiki/Dark_Shard] and 21 of any [link evil substance | | Recipes.html].

A full set of Necromancer armor provides the following benefits:

- +31 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +25% [link summon | | Summon_Weapons.html] damage
- +15% summon [link attack speed | | https://terraria.wiki.gg/wiki/Attack_speed]
- +1 [link minion | | https://terraria.wiki.gg/wiki/Summon_weapons#Minions] slot
- +25% [link artifact | | Artifacts.html] damage
Set bonus
- Slaying an enemy provides a stackable, temporary boost to all stats
- [link Mana | | https://terraria.wiki.gg/wiki/Mana] usage is halved when in a [link Graveyard | | https://terraria.wiki.gg/wiki/Graveyard]
- Artifact minion mana costs are halved
- Enemies spawn more frequently
- +3 minion slots

Chambersite Gemstones

[statblock Chambersite_Gemstone]

Chambersite Gemstones are a type of [link Gemstone | | https://terraria.wiki.gg/wiki/Gemstone_Blocks] [link block | | Blocks.html] found in the [link Underground | | https://terraria.wiki.gg/wiki/Underground] and [link Cavern | | https://terraria.wiki.gg/wiki/Cavern] layers underneath the [link world evil | | https://terraria.wiki.gg/wiki/Evil_biomes]. When any are mined, they will drop a [link Chambersite] [link gem | | https://terraria.wiki.gg/wiki/Gems] similarly to the other types. They can be mined by any [link pickaxe | | https://terraria.wiki.gg/wiki/Pickaxes] or [link drill | | https://terraria.wiki.gg/wiki/Drills], or destroyed by [link explosives | | Demolitionist_Weapons.html].

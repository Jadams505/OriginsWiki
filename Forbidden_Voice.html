Forbidden Voice

[statblock Forbidden_Voice]

The Forbidden Voice is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] dropped by the [link Treasure Bag | | https://terraria.wiki.gg/wiki/Treasure_Bag] dropped by the [link Eater of Worlds | | https://terraria.wiki.gg/wiki/Eater_of_Worlds] in [link Master Mode | | https://terraria.wiki.gg/wiki/Master_Mode]. While the [link cursed | | Curses.html] version is equipped, the player is [link Silenced | | https://terraria.wiki.gg/wiki/Silenced], but the mouth will shout at enemies dealing considerable damage and [link knockback | | https://terraria.wiki.gg/wiki/Knockback]. While the normal version is equipped, just allows the player to spit at enemies inflicting [link Weak | | https://terraria.wiki.gg/wiki/Weak] and moderate damage. Both versions use up the player's [link mana | | https://terraria.wiki.gg/wiki/Mana], but the cursed version using up more.

Forbidden Voice is a [link lore item | | Lore_Item.html] indicated by "You hear whispers nearby..."

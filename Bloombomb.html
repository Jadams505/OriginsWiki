Bloombomb

[statblock Bloombomb]

The Bloombomb is an [link explosive | | Demolitionist_Weapons.html] that can be crafted using [link any Seeds | | https://terraria.wiki.gg/wiki/Seeds] and [link Bombs | | https://terraria.wiki.gg/wiki/Bomb] or pruchased from the [link Dryad | | https://terraria.wiki.gg/wiki/Dryad] [link NPC | | NPCs.html] after completing her 'Clearing the Way' [link quest | | Quests.html]. Upon detonation, the Bloombomb will release 11-14 seeds that will bounce around briefly. It does not destroy tiles or any vegetation from its explosion.

Tips

- It is far cheaper to buy Bloombombs directly from the Dryad as opposed to crafting them, which will save the player [price 3|0] for every Bloombomb bought.
- Bloombombs are best used in enclosed areas where the seeds don't travel far away from the bomb's location.

Cryosten Armor

[statblock Cryosten_Armor]

Cryosten armor is an [link armor] set consisting of the Cryosten Helmet, Cryosten Breastplate and Cryosten Greaves. In order to craft the full set the player will need 135 [link Ice blocks | | https://terraria.wiki.gg/wiki/Ice_Block], 3 [link Life Crystals | | https://terraria.wiki.gg/wiki/Life_Crystal] and 12 [link Shiverthorn | | https://terraria.wiki.gg/wiki/Shiverthorn].

A full set of Cryosten armor provides the following benefits:

- +7 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +1 HP/s [link life regeneration | | https://terraria.wiki.gg/wiki/Health_regeneration]
- +12% max [link life | | https://terraria.wiki.gg/wiki/Health]
- +5% movement speed
- Increased movement speed on ice
Set bonus
- +1.5 HP/s for 60 ticks after picking up [link Hearts | | https://terraria.wiki.gg/wiki/Heart]
- Immunity to [link Chilled | | https://terraria.wiki.gg/wiki/Chilled] and [link Frozen | | https://terraria.wiki.gg/wiki/Frozen]

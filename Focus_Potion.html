Focus Potion

[statblock Focus_Potion]

The Focus Potion is a [link buff potion | | Buff_Potions.html] that can be crafted using [link Bottled Water | | https://terraria.wiki.gg/wiki/Bottled_Water], [link Fallen Star | | https://terraria.wiki.gg/wiki/Fallen_Star] and [link Brineglow]. When consumed, it provides the Mana Channeling [link buff | | Buffs.html], which causes the player to consume [link mana | | https://terraria.wiki.gg/wiki/Mana] to boost weapon damage. This effect will last for 4 minutes, but can be canceled anytime by right-clicking the icon. The effect still works with [link magic weapons] and just consumes more mana.

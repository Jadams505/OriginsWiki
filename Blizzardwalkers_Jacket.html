Blizzardwalker's Jacket

[statblock Blizzardwalkers_Jacket]

The Blizzardwalker's Jacket is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] that is dropped by [link Deerclops | | https://terraria.wiki.gg/wiki/Deerclops] in [link Master Mode | | https://terraria.wiki.gg/wiki/Master_Mode]. While this is equipped, the player will dodge the next attack as long as there have been no enemies within 15 tiles of the player for 5 seconds.

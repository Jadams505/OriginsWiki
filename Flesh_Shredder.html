Flesh Shredder

[statblock Flesh_Shredder]

The Flesh Shredder is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link crafting station | | Crafting_Stations.html] used to craft [link furniture] based on the [link Riven | | Riven_Hive.html]-themed [link Residue Block]. The Flesh Shredder can be purchased from the [link Steampunker | | https://terraria.wiki.gg/wiki/Steampunker] for [price 10|0|0] but only appears for sale in worlds that are generated with the Riven.

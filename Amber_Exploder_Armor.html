Amber Exploder Armor

[statblock Amber_Armor]

Amber Exploder armor is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link armor] set consisting of the Amber Exploder Goggles, Amber Exploder Resin and Amber Exploder Guards. In order to craft the full set the player will need 24 [link Amber | | https://terraria.wiki.gg/wiki/Amber], 9 [link Souls of Night | | https://terraria.wiki.gg/wiki/Soul_of_Night] and 72 [link Carburite].

A full set of Amber Exploder armor provides the following benefits:

- +21 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- -20% [link self-damage | | Self_Damage.html]
- +10% movement speed
Set bonus
- -38% [link explosive | | Demolitionist_Weapons.html] blast radius
- All explosives are preserved in an amber resin and release damaging amber shards upon detonation
- Amber shards inflict [link Slow | | https://terraria.wiki.gg/wiki/Slow], halve enemy [link defense | | https://terraria.wiki.gg/wiki/Defense] and heal the player for any [link life | | https://terraria.wiki.gg/wiki/Health] lost from self-damage

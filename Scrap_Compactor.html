Scrap Compactor

[statblock Scrap_Compactor]

The Scrap Compactor is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] that is obtained from the [link Treasure Bag | | https://terraria.wiki.gg/wiki/Treasure_Bag] dropped by the [link Scrapper] in [link Master Mode | | https://terraria.wiki.gg/wiki/Master_Mode]. While equipped, all [link explosives | | Demolitionist_Weapons.html] will explode into frenzied shrapnel. Additionally, any nearby enemies that are below 40% of their total health and are currently inflicted with [link Impeding Shrapnel] will instantly die (this effect does not apply to [link bosses | | https://terraria.wiki.gg/wiki/Bosses]).

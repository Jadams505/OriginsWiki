Primordial Permafrost

[statblock White_Ice]

Primordial Permafrost is a [link block | | Blocks.html] type found in a [link Snow biome | | https://terraria.wiki.gg/wiki/Snow_biome] converted by the [link Riven | | Riven_Hive.html]. It is the Riven-equivalent to [link ice blocks | | https://terraria.wiki.gg/wiki/Ice_Blocks], and can be converted to other variants with [link any thrown powder | | https://terraria.wiki.gg/wiki/Thrown_Powder].

Mine Flayer

[statblock Mine_Flayer]

The Mine Flayer is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link mine | | Mines.html] [link launcher | | Launchers.html] crafted using mech components dropped by the [link mechanical bosses | | https://terraria.wiki.gg/wiki/Mechanical_bosses]. When this weapon is swung, it will rapidly release 9 [link mines] in a large arc at a great velocity.

Its best [link modifier | | Modifiers.html] is Guileless.

- It is most effective the closer the player is to the target, and while some form of [link self-damage | | Self_Damage.html] protection is in use. This is so the player can prevent each of
the released mines from being too spread out.

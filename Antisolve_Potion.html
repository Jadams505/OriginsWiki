Antisolve Potion

[statblock Antisolve_Potion]

The Antisolve Potion is a [link buff potion | | Buff_Potions.html] that can be crafted using [link Bottled Brine], [link Brineglow] and [link Peat Moss]. When consumed, it provides the Unsolvable [link buff | | Buffs.html], which provides immunity to the [link Toxic Shock] and [link Torn] debuffs. This effect will last for 8 minutes, but can be canceled anytime by right-clicking the icon.

Tips

- A safe way to obtain the Antisolve Potion without dying is to use a projectile to cut off any Brineglow vines exposed from the drop-off just before the
[link Brine Pool], drop down in the water with a [link Recall Potion | | https://terraria.wiki.gg/wiki/Recall_Potion] or [link Magic Mirror | | https://terraria.wiki.gg/wiki/Magic_Mirrors] to pick up the fallen Brineglow, and then simply craft it afterwards. It is
advisable to collect the Bottled Brine using platforms and Peat Moss from some enemies found in the [link Underground Jungle | | https://terraria.wiki.gg/wiki/Underground_Jungle] before returning back home
to save time.

Nanite Armor

[statblock Nanite_Armor]

Nanite armor is a post-[link Moon Lord | $default | https://terraria.wiki.gg/wiki/Moon_Lord] [link armor] set. It has melee, ranged, magic, summoner and explosive set versions, depending on the equipped headpiece. In order to obtain a full, individual set the player will need to complete the [link Test of Ancients] [link event | $default | Events.html] at least 3 times, however, the event will need to be completed at least 7 times for all pieces.

A full set of Nanite armor provides the following benefits:

Nanite Helmet
- +44 [link defense | $default | https://terraria.wiki.gg/wiki/Defense]
- +25% [link melee | $default | Melee_Weapons.html] weapon size

Nanite Tarnhelm
- +41 defense
- +40% [link ammo conservation | $default | https://terraria.wiki.gg/wiki/Ammo_conservation]

Nanite Hood
- +38 defense
- -40% [link mana | $default | https://terraria.wiki.gg/wiki/Mana] consumption

Nanite Crown
- +35 defense
- +3 [link minion | $default | https://terraria.wiki.gg/wiki/Summon_weapons#Minions] slots

Nanite Globe
- +32 defense
- +50% [link explosive | $default | Demolitionist_Weapons.html] [link velocity | $default | https://terraria.wiki.gg/wiki/Velocity]

Set bonus
- Opens up an slot to install [link Nanite Cores]

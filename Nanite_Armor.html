Nanite Armor

[statblock Nanite_Armor]

Nanite armor is a post-[link Moon Lord | | https://terraria.wiki.gg/wiki/Moon_Lord] [link armor] set consisting of the Nanite Mask, Nanite Coat and Nanite Pants. In order to craft the full set the player will need 60 [link Material-36 Bundles | | Material_36_Bundle.html].

A full set of Nanite armor provides the following benefits:

- +55 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +15% [link magic | | Magic_Weapons.html] damage
- +100 [link mana | | https://terraria.wiki.gg/wiki/Mana] capacity
Set bonus
- [link Critical hits | | https://terraria.wiki.gg/wiki/Critical_hit] instead triple damage
- Every 6 mana regenerated heals 1 [link health | | https://terraria.wiki.gg/wiki/Health] - Grants the 'Chameleon' ability by pressing the ability key

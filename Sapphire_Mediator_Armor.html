Sapphire Mediator Armor

[statblock Sapphire_Armor]

Sapphire Mediator armor is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode], post-[link Mech | | https://terraria.wiki.gg/wiki/Mechanical_bosses] [link armor] set consisting of the Sapphire Mediator Hood, Sapphire Mediator Vest and Sapphire Mediator Tights. In order to craft the full set the player will need 24 [link Sapphires | | https://terraria.wiki.gg/wiki/Sapphire], 9 [link Souls of Might | | https://terraria.wiki.gg/wiki/Soul_of_Might] and 72 [link Carburite].

A full set of Sapphire Mediator armor provides the following benefits:

- +21 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- -40 [link mana | | https://terraria.wiki.gg/wiki/Mana]
- +15% [link magic | | Magic_Weapons.html] damage
- +10% movement speed
Set bonus
- Generate a massive aura that pushes away enemies and projectiles with a force depending on how close they are to the player
- Allies receive a minor boost in all stats when within the aura
- The aura fades when the player runs out of mana

Acrid Handcannon

The Acrid Handcannon is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link handcannon | | Handcannons.html] crafted with [link Eitrite Bars | | Eitrite_Bar.html]. It behaves like other handcannons except that its slugs also release 3-5 brine droplets upon detonation.

Its best [link modifier | | Modifiers.html] is Guileless.

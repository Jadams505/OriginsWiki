Air Tank

The Air Tank is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link accessory | | Accessories.html] crafted using either Adamantite Bars or Titanium Bars with [link Rubber]. While worn, it provides the player with immunity to [link Suffocation | | https://terraria.wiki.gg/wiki/Suffocation] and an additional 25 seconds of [link breath | | https://terraria.wiki.gg/wiki/Breath_meter].

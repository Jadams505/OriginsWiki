Blotopus

[statblock Blotopus]

The Blotopus is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link gun | | Guns.html] that can be [link fished | | https://terraria.wiki.gg/wiki/Fishing] for in [link the Crimson | | https://terraria.wiki.gg/wiki/Crimson]. It is the Crimson-equivalent to [link Corruption's | | https://terraria.wiki.gg/wiki/The_Corruption] [link Purple Clubberfish | | https://terraria.wiki.gg/wiki/Purple_Clubberfish]. It rapidly fires needles out of its mouth that are affected by gravity and inflict [link Bleeding | | https://terraria.wiki.gg/wiki/Bleeding].

Its best [link modifier | | Modifiers.html] is Demonic or Deadly.

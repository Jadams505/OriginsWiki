Lost Armor

[statblock Lost_Armor]

Lost armor is an [link armor] set consisting of the Lost Monocle, Lost Cloak and Lost Pants. In order to craft the full set the player will need 42 [link Lost Ore] and 66 [link Strange String].

A full set of Lost armor provides the following benefits:

- +10 [link defense | | https://terraria.wiki.gg/wiki/Defense] - +2 [link mana | | https://terraria.wiki.gg/wiki/Mana] regeneration - +15% [link magic | | Magic_Weapons.html] damage - +5% movement speed Set bonus - +15% [link mana shielding]

Notes

- Lost Armor's effect is stackable with other mana shielding boosts (see [link item synergies | | Item_Synergies#Mana_Shielding.html]). - Mana shielding takes into account the damage the player would have received with 0 defense and 0% damage reduction. This results in the proper amount of damage reduced from the attack, but an inefficiency in the mana consumed from it. - Whenever the player benefits from mana shielding, [link Defiled Exhaustion] will be applied preventing any mana regneration for 3.2 seconds.

Tips

- Lost armor pairs well with [link Magic Cuffs | | https://terraria.wiki.gg/wiki/Magic_Cuffs] and reduces the amount of mana consumed from mana shielding by 66%.

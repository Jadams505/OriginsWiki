Futurephones

[statblock Futurephones]

Futurephones are an [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode], post-[link Mechanical | | https://terraria.wiki.gg/wiki/Mechanical_bosses] [link accessory | | Accessories.html] combining [link Headphones] and [link Sniper's Mark | | Snipers_Mark.html]. While equipped, all attacks (other than from a [link minion | | Minions.html]) will be able to [link tag | | https://terraria.wiki.gg/wiki/Whips] enemies. Additionally, [link generic damage] is increased by 5% and generic [link critical strike chances | | https://terraria.wiki.gg/wiki/Critical_hit] by 10% against tagged enemies.

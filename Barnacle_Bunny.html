Barnacle Bunny

[statblock Barnacle_Bunny]

The Barnacle Bunny is an [link enemy | | Enemies.html] that has been corrupted, either by the player using [link Sentient Powder] or by a [link Blood Moon | | https://terraria.wiki.gg/wiki/Blood_Moon] event on a [link Riven | | Riven_Hive.html] world. The Barnacle Bunny behaves like its vanilla counterparts: the [link Corrupt Bunny | | https://terraria.wiki.gg/wiki/Corrupt_Bunny] and [link Vicious Bunny | | https://terraria.wiki.gg/wiki/Vicious_Bunny].

Notes

- Barnacle Bunnies can be farmed for the [link Bunny Hood | | https://terraria.wiki.gg/wiki/Bunny_Hood] and [link Barnacle Bunny Banner | | Banners.html] using a [link Bunny Statue | | https://terraria.wiki.gg/wiki/Statues#Bunny_Statue], [link Pressure Plate | | https://terraria.wiki.gg/wiki/Pressure_Plates], [link Sentient Powder] and [link Wire | | https://terraria.wiki.gg/wiki/Wire]. However, these statue-spawned ones do not drop [link coins | | https://terraria.wiki.gg/wiki/Coins].
- Although they spawn during the Blood Moon event, they cannot drop [link Bloody Tears | | https://terraria.wiki.gg/wiki/Bloody_Tear].

Trivia

- The [link Bestiary | | https://terraria.wiki.gg/wiki/Bestiary] entry for the Barnacle Bunny: "Bunnies are pure beings that cannot normally adapt to the Riven Hive's ecosystem, but will be easily capable of under the effects of a Blood Moon."

Nerve Impulse Manipulator

[statblock Nerve_Impulse_Manipulator]

The Nerve Impulse Manipulator is a boss-summoning [link consumable | | Consumables.html] item crafted using [link Dreadful Powder] and [link Strange String]. Upon use, it will summon the [link Defiled Amalgamation] anywhere in the world. If not used in the [link Defiled Wastelands], it will cause the Defiled Amalgamation to automatically enter its "enraged" state when spawned.

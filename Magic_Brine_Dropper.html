Magic Brine Dropper

[statblock Magic_Brine_Dropper]

The Magic Brine Dropper is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] item used to place "brine emissions" on ceilings. The smoky effect is purely visual, and has no functional use. The Magic Brine Dropper is crafted with an [link Empty Dropper | | https://terraria.wiki.gg/wiki/Empty_Dropper] at [link Brine Water | | Brine_Pool.html], with the [link Crystal Ball | | https://terraria.wiki.gg/wiki/Crystal_Ball] nearby. It smoke effect can be removed by destroying the block its was place under, but will not drop any item.

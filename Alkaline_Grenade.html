Alkaline Grenade

[statblock Alkaline_Grenade]

The Alkaline Grenade is a type of [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link explosive | | Demolitionist_Weapons.html] that can be crafted using [link Grenades | | https://terraria.wiki.gg/wiki/Grenade] and [link Bottled Brine] or purchased from the [link Demolitionist | | https://terraria.wiki.gg/wiki/Demolitionist] [link NPC | | https://terraria.wiki.gg/wiki/NPCs] after selling 75 [link Peat Moss] to him once Hardmode is reached. The alkaline Grenade explodes into 3-5 brine droplets that each inflict [link Toxic Shock] lasting anywhere from 4.5 to 6 seconds.

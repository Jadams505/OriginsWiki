Mythic Armor

[statblock Mythic_Armor]

Mythic armor is an [link armor] set consisting of the Mythic Skull, Mythic Shell and Mythic Leggings. In order to craft the full set the player will need 50 [link Bones | | https://terraria.wiki.gg/wiki/Bone], 30 [link Clouds | | https://terraria.wiki.gg/wiki/Cloud] and 30 [link Feathers | | https://terraria.wiki.gg/wiki/Feather].

A full set of Mythic armor provides the following benefits:

- +12 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +15% [link explosive | | Demolitionist_Weapons.html] [link velocity | | https://terraria.wiki.gg/wiki/Velocity]
- +45% jump speed
- +20% movement speed
Set bonus
- Increased explosive damage by 20% of the player's current speed
- Increased explosive velocity by 50% of the player's current speed

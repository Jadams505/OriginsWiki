Construction Core

[statblock Construction_Core]

The Construction Core is a post-[link Moon Lord | | https://terraria.wiki.gg/wiki/Moon_Lord], Eyndum-tier [link accessory | | Accessories.html] that is crafted from [link materials] obtained through the [link Test of the Ancients] [link event | | Events.html] at an [link Elderforge]. It can only be equipped in a dedicated core slot provided by a full set of [link Eyndum Armor] that appears to the left of the player's equipment section. It provides the following boosts:

- +100% tile placement speed
- +150% wall placement speed
- +200% [link mining speed | | https://terraria.wiki.gg/wiki/Mining_speed]
- +30 tile range
- +40 block placement range

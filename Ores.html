Ores

If you are looking for the official page of vanilla ores, click [link here | | https://terraria.wiki.gg/wiki/Ores].

Ores are primary crafting [link materials] necessary for the game to progress. They generally must be crafted smelted into [link bars] at a [link Furnace | | https://terraria.wiki.gg/wiki/Furnace], after which they can then be used to craft several essential [link tools], [link weapons], [link armor] and other [link items | | https://terraria.wiki.gg/wiki/Items]. Some items (especially in Origins) require raw ores themselves, most commonly [link bricks], [link armors | | Armor.html] and [link potions | | Potion.html]. Raw ores can also be used as plain blocks for construction.

Ores generally appear as veins of placed [link blocks] in the [link Underground | | https://terraria.wiki.gg/wiki/Underground] [link layer | | https://terraria.wiki.gg/wiki/Layers] and below (or in [link Floating Islands | | https://terraria.wiki.gg/wiki/Floating_Island] like [link Felnum Ore]), and are mined with a [link pickaxe], [link drill] or [link explosives | | Demolitionist_Weapons.html]. Lower-tier ores appear inside [link Dirt | | https://terraria.wiki.gg/wiki/Dirt_Block] near the [link surface | | https://terraria.wiki.gg/wiki/Layers#Surface], or within Floating Islands. Some ores are also dropped by [link bosses] like the [link Eye of Cthulhu | | https://terraria.wiki.gg/wiki/Eye_of_Cthulhu], [link world evil | | Evil_Biomes.html] bosses and [link Moon Lord | | https://terraria.wiki.gg/wiki/Moon_Lord].

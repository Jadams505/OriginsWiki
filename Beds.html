Beds

If you are looking for the official page of vanilla beds, click [link here | | https://terraria.wiki.gg/wiki/Beds].

[statblock Beds]

Beds are functional [link furniture] items which allow the player to set a new [link spawn | | https://terraria.wiki.gg/wiki/Spawn] point by pressing the Open/Activate button on the foot half of the bed. This requires it to be placed in a mostly valid [link house | | https://terraria.wiki.gg/wiki/House]. Beds furthermore allow the player to sleep in them by pressing the Open/Activate button on the head half of the bed. This makes time pass five times faster and gives a boost to [link health regeneration | | https://terraria.wiki.gg/wiki/Health_regeneration], similar to [link chairs] and [link sofas]. Beds fulfill the [link comfort item | | https://terraria.wiki.gg/wiki/House#Comfort_items] requirement for purposes of [link NPC | | NPCs.html] housing.

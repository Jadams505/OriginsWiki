Broken Record

[statblock Broken_Record]

The Broken Record is a boss-summoning [link consumable | | Consumables.html] item crafted using [link Urns of Ashes | | Urn_of_Ashes.html] and [link Biocomponent 10]. It does not function like other boss-summoning items since its respective boss, the [link Scrapper], is currently not implemented. Instead it acts like a grab bag upon its use, dropping one of the following items:

- [link Ace Shrapnel] accompanied by 200 [link Scrap]
- [link Area Denial]
- [link Neural Network] accompanied by 100 [link Musket Balls | | https://terraria.wiki.gg/wiki/Musket_Ball]
- [link Seal of Cinders]
- [link Smith's Hammer | | Smiths_Hammer.html]

Chromtain Armor

Chromtain armor is a post-Moon Lord armor set consisting of the Chromtain Helmet, Chromtain Breastplate and Chromtain Greaves. In order to craft the full set the player will need 60 Chromtain Bars. Chromtain is currently the highest defense armor set with Origins.

A full set of Chromtain armor provides the following benefits:

- +81 defense
- +20 max life
Set bonus
- +400 aggro
- Immunity to Broken Armor

Chromtain Armor

[statblock Chromtain_Armor]

Chromtain armor is a post-[link Moon Lord | | https://terraria.wiki.gg/wiki/Moon_Lord] [link armor] set consisting of the Chromtain Helmet, Chromtain Breastplate and Chromtain Greaves. In order to craft the full set the player will need 60 [link Chromtain Bars | | Chromtain_Bar.html]. Chromtain is currently the highest [link defense | | https://terraria.wiki.gg/wiki/Defense] armor set with Origins.

A full set of Chromtain armor provides the following benefits:

- +81 defense
- +20 max [link life | | https://terraria.wiki.gg/wiki/Health]
Set bonus
- +400 [link aggro]
- Immunity to [link Broken Armor | | https://terraria.wiki.gg/wiki/Broken_Armor]

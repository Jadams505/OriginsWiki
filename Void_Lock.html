Void Lock

[statblock Void_Lock]

The Void Lock is a [link consumable | | Consumables.html] item that is crafted from a [link Chest Lock | | https://terraria.wiki.gg/wiki/Chest_Lock], [link Bones | | https://terraria.wiki.gg/wiki/Bone], [link Jungle Spores | | https://terraria.wiki.gg/wiki/Jungle_Spore] and any [link evil tissue | | Recipes.html]. Pressing the Use button on certain types of [link Chests] while holding it will turn the Chest into a locked Chest and consume the Void Lock. Chests locked by this can only be opened by the player that placed the Void Lock on them, providing security from griefing in [link multiplayer | | https://terraria.wiki.gg/wiki/Multiplayer]. The chest can be unlocked by using the Interact button without any item being needed.

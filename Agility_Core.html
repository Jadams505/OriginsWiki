Agility Core

[statblock Agility_Core]

The Agility Core is a post-[link Moon Lord | | https://terraria.wiki.gg/wiki/Moon_Lord], Eyndum-tier [link accessory | | Accessories.html] that is crafted from [link materials] obtained through the [link Test of the Ancients] [link event | | Events.html] at an [link Elderforge]. It can only be equipped in a dedicated core slot provided by a full set of [link Eyndum Armor] that appears to the left of the player's equipment section. It provides the following increases:

- +100% [link flight time | | https://terraria.wiki.gg/wiki/Flight] - +300% movement speed - +200% acceleration/deceleration speed - +200% [link running speed | | https://terraria.wiki.gg/wiki/Sprinting_accessories] - +5 jump speed

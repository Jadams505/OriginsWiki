Meteor

[statblock Meteor]

The Meteor is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link launcher | | Launchers.html] that can be crafted using [link Meteorite Bars | | https://terraria.wiki.gg/wiki/Meteorite_Bar]. The Meteor converts its [link ammunition] into miniature meteorites that are always shot upwards regardless of where the player aims in order to reach the targeted area. Upon impact with an enemy or tile, the meteorite will explode initially but remain in place wherever it landed for 3 seconds before exploding again and destroying tiles similarly to [link Bombs | | https://terraria.wiki.gg/wiki/Bomb]. The Meteor additionally has guaranteed [link critical hits | | https://terraria.wiki.gg/wiki/Critical_hit] against flying enemies while it is falling.

Its best [link modifier | | Modifiers.html] is Guileless for any other statistic than [link velocity | | https://terraria.wiki.gg/wiki/Velocity]. For velocity, Heavy-duty is recommended so the projectile doesn't have to travel as high up to reach its target destination, which is particularly ideal for enclosed spaces.

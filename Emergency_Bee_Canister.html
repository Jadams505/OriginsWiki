Emergency Bee Canister

[statblock Emergency_Bee_Canister]

The Emergency Bee Canister is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] that is obtained by the [link Treasure Bag | | https://terraria.wiki.gg/wiki/Treasure_Bag] dropped by [link Queen Bee | | https://terraria.wiki.gg/wiki/Queen_Bee] in [link Master Mode | | https://terraria.wiki.gg/wiki/Master_Mode]. When equipped, [link enemy Bees | | https://terraria.wiki.gg/wiki/Bee] become friendly, all bees have a 7% chance to be duplicated and the player can freely move through [link honey | | https://terraria.wiki.gg/wiki/Honey].

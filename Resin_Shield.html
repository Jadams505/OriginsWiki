Resin Shield

[statblock Resin_Shield]

The Resin Shield is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link shield | | https://terraria.wiki.gg/wiki/Shields] [link accessory | | Accessories.html] created with [link Amber | | https://terraria.wiki.gg/wiki/Amber], the [link Obsidian Shield | | https://terraria.wiki.gg/wiki/Obsidian_Shield] and [link Carburite]. It provides immunity to [link knockback | | https://terraria.wiki.gg/wiki/Knockback] and the [link Burning | | https://terraria.wiki.gg/wiki/Burning] debuff while also periodically blocking all [link self-damage | | Self_Damage.html] on a 5 second cooldown. The cooldown triggers anytime the player would have recieved an instance of self-damage. The Resin Shield loses its color whenever it is on cooldown.

Resin Shield

[statblock Resin_Shield]

The Resin Shield is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] shield [link accessory | | Accessories.html] created with Amber, the Obsidian Shield and [link Carburite]. It provides immunity to knockback and the Burning debuff while also periodically blocking all [link self-damage | | Self_Damage.html] on a 5 second cooldown. The cooldown triggers anytime the player would have recieve an instance of self-damage. The Resin Shield loses its color when it is on cooldown.

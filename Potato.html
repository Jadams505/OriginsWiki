Potato

[statblock Potato]

The Potato is a [link food] item that has a 1/34 (2.94%) chance to drop from enemies during the [link Goblin Army | | https://terraria.wiki.gg/wiki/Goblin_Army] [link event | | Events.html], 1/13 chance to be dropped by any [link zombie | | https://terraria.wiki.gg/wiki/Zombie] or [link Harpy | | https://terraria.wiki.gg/wiki/Harpy], and is guaranteed to drop from [link Nymphs | | https://terraria.wiki.gg/wiki/Nymph]. When consumed, it provides the [link Well Fed | | https://terraria.wiki.gg/wiki/Well_Fed] [link buff | | Buffs.html], which lasts for 10 minutes. It can be placed by right-clicking above any [link soils].

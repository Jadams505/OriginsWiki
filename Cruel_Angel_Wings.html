Cruel Angel Wings

[statblock Cruel_Angel_Wings]

Cruel Angel Wings are an [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode], post-[link Plantera | | https://terraria.wiki.gg/wiki/Plantera] [link accessory | | Accessories.html] crafted using [link Angel Wings | | https://terraria.wiki.gg/wiki/Wings#Angel_Wings], [link Ectoplasm | | https://terraria.wiki.gg/wiki/Ectoplasm] and [link Handy Helper]. When equipped, the player will be able to temporarily fly while holding the Jump key like any other [link wings]. Additionally, the player will have any incoming projectiles and enemies knocked back that are within a 4.4-tile (or 71-unit) radius of the player while in flight or falling. While Handy Helper is equipped, the cooldown of deflections will be reduced by 50%.

Cruel Angel wings have a flight time of 2.17 seconds and a max horizontal speed of 38.

Riven Jungle Grass is a [link block | | Blocks.html] type found in a [link Jungle | | https://terraria.wiki.gg/wiki/Jungle_Biome] converted by the [link Riven | | Riven_Hive.html]. It is the Riven-equivalent to [link Jungle Grass | | https://terraria.wiki.gg/wiki/Grasses#Jungle_grass], and can be converted to other variants with [link any thrown powder | | https://terraria.wiki.gg/wiki/Thrown_Powder]. Riven Jungle Grass can be placed and spread using [link Riven Seeds]. Given enough time, [link foliage | | Plants.html] will start to grow on top of Riven Jungle Grass.

Blast Armor

[statblock Blast_Armor]

Blast armor is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode], post-[link Mech | | https://terraria.wiki.gg/wiki/Mechanical_bosses] [link armor] set consisting of the Blast Helmet, Blast Plate and Blast Guards. In order to craft the full set the player will need 54 [link Chlorophyte Bars | | https://terraria.wiki.gg/wiki/Chlorophyte_Bar], 3 [link Blast-Resistant Plates | | Blast_Resistant_Plate.html], 6 [link Busted Servos | | Busted_Servo.html], 4 [link Rotors | | Rotor.html] and 4 [link Power Cores | | Power_Core.html].

A full set of Blast armor provides the following benefits:

- +39 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +10% [link generic | | Generic_Damage.html] [link critical strike | | https://terraria.wiki.gg/wiki/Critical_hit] chance
- -15% [link self-damage | | Self_Damage.html]
- Immunity to [link knockback | | https://terraria.wiki.gg/wiki/Knockback]
- +25% movement speed
- +2% acceleration/deceleration
- +20% jump speed
Set bonus
- -25% self-damage
- Allows the player to berserk by pressing the ability key
- 80% of incoming damage and self-damage increases the berserk duration
- All [link explosive | | Demolitionist_Weapons.html] combat stats are increased by 50% for berserk duration
- The helmet changes its appearance while berserked

Ability

Berserk
Consumes 14 berserk count every second.
- Increases explosive [link velocity | | https://terraria.wiki.gg/wiki/Velocity], blast radius, [link attack speed | | https://terraria.wiki.gg/wiki/Attack_speed], [link knockback | | https://terraria.wiki.gg/wiki/Knockback], and reduces link fuse-time | | Fuse_Time.html] by 50%
- Increases explosive critical strike chance of the held weapon by 50% of its current chance

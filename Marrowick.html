Marrowick

[statblock Marrowick]

Marrowick is a type of [link wood | | https://terraria.wiki.gg/wiki/Woods] [link block | | Blocks.html] and [link material | | Materials.html] dropped by [link Exoskeletal Trees] found in the [link Riven Hive]. Marrowick behaves the same as regular wood, but adopts a light yellow coloration with a smoother consistency. Exoskeletal Trees can be planted anew by planting [link Acorns | | https://terraria.wiki.gg/wiki/Acorn] on [link Riven Grass].

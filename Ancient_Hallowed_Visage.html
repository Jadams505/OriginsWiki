Ancient Hallowed Visage

[statblock Ancient_Hallowed_Visage]

The Ancient Hallowed Visage is a helmet [link armor] piece that reproduces the pre-release design, and is otherwise identical in all ways to the current [link Hallowed Visage]. Each piece is interchangeable with the equivalent [link Hallowed armor | | https://terraria.wiki.gg/wiki/Hallowed_armor] piece, while still granting the same set bonus.

Ancient Hallowed Visage

[statblock Ancient_Hallowed_Visage]

The Ancient Hallowed Visage is a helmet armor piece that reproduces the pre-release design, and is otherwise identical in all ways to the current Hallowed Visage. Each piece is interchangeable with the equivalent Hallowed armor piece, while still granting the same set bonus.

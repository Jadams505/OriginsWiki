Absorption Potion

[statblock Absorption_Potion]

The Absorption Potion is a [link buff potion | | Buff_Potions.html] that can be crafted using [link Bottled Water | | https://terraria.wiki.gg/wiki/Bottled_Water], a [link Bomb Fish | | https://terraria.wiki.gg/wiki/Bomb_Fish] and a [link Fallen Star | | https://terraria.wiki.gg/wiki/Fallen_Star]. When consumed, it provides the Safe From Yourself [link buff | | Buffs.html], which provides immunity to all [link self-damage | | Self_Damage.html] to the player from one of their own [link explosives | | Demolitionist_Weapons.html]. This effect will last for 6 minutes, but can be canceled anytime by Interact the icon. Unlike most self-damage boosts, it does not affect any self-damage from other sources.

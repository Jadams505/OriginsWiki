Bleeding Obsidian Kunai

[statblock Bleeding_Obsidian_Kunai]

The Bleeding Obsidian Kunai is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] throwing weapon that acts as a [link ranged | | Ranged_Weapons.html] counterpart to the [link Magic Dagger | | https://terraria.wiki.gg/wiki/Magic_Dagger]. The Bleeding Obsidian Kunai is a [link consumable | | Consumables.html] weapon that replenishes itself after the thrown projectile is destroyed. The Kunai can be thrown more often depending on how many stacks of this item the player has, however, only 8 stacks of this item can be reached max. Enemies struck by this weapon are inflicted [link Bleeding | | https://terraria.wiki.gg/wiki/Bleeding] for 5 seconds, [link Cursed Inferno | | https://terraria.wiki.gg/wiki/Cursed_Inferno] for 2 seconds, [link Ichor | | https://terraria.wiki.gg/wiki/Ichor_(debuff)] for 3 seconds and [link Torn] at 10% severity for 5 seconds.

True Ravel

[statblock True_Ravel]

True Ravel is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode], post-[link Plantera | | https://terraria.wiki.gg/wiki/Plantera] [link accessory | | Accessories.html] that is crafted using all of [link Ravel]'s upgrades. True Ravel turns the player into a small sphere that provides slightly more speed than their normal form and gives a 10% chance to dodge attacks. On top of the standard ravel statistics, True Ravel also grants the following:

Anytime: - The ability to dash While ravelled: - Increased movement speed and acceleration - Generate a kinetic wave that damages and knocks back enemies at top speed - +18 seconds of [link breath | | https://terraria.wiki.gg/wiki/Breath_meter] - Grants the ability to [link swim | | https://terraria.wiki.gg/wiki/Swimming] - Emit an aura of light - Grants usage of any [link pickaxe] or [link drill] in the player's [link inventory | | https://terraria.wiki.gg/wiki/Inventory] - Grants usage of equipped [link hook | | https://terraria.wiki.gg/wiki/Hooks] - -400 [link aggro] - Grants the ability to attach to walls and ceilings using movement keys (see [link Spider Ravel]) - Grants prolonged flight and immunity to [link fall damage | | https://terraria.wiki.gg/wiki/Fall_damage]

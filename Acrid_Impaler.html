Acrid Impaler

[statblock Acrid_Impaler]

The Acrid Impaler is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link harpoon gun | | Harpoon_Guns.html] crafted with [link Eitrite Bars | | Eitrite_Bar.html]. All harpoons fired by this weapon act like an [link Alkaline Harpoon], regardless of the harpoon type used as [link ammunition].

Its best [link modifiers] are Godly and Demonic.

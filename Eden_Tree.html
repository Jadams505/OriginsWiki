Eden Tree

[statblock Eden_Tree]

Eden Trees are a variant of [link trees | | https://terraria.wiki.gg/wiki/Trees] grown on [link Edengrass] [link blocks] found at [link the Dawn]. Eden Trees require an [link axe | | Axes.html] or [link chainsaw | | Chainsaws.html] to destroy and drop [link Edenwood], which can be used to craft [link Edenwood armor]. Eden Trees currently do not have a [link fruit | | https://terraria.wiki.gg/wiki/Fruits] drop obtained by [link shaking | | https://terraria.wiki.gg/wiki/Trees#Shaking] them.

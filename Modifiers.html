Modifiers

If you are looking for the official page of vanilla modifiers, click [link here | | https://terraria.wiki.gg/wiki/Modifiers].

A Modifier, also known as a Prefix, applies permanent changes to an item's statistics. It changes the name of an item by adding a prefix to the item's name. Origins adds new tooltips with the new [link explosives | | Demolitionist_Weapons.html] added and some post-[link Moon Lord | | https://terraria.wiki.gg/wiki/Moon_Lord] modifiers, which can be selectively reforged by the [link Cubekon Tinkerer] [link NPC | | NPCs.html]. Modifiers added by Origins can be normally obtained with the exception of the aforementioned post-Moon Lord modifiers.

Reforging

Reforging can now be done by the [link Goblin Tinkerer | | https://terraria.wiki.gg/wiki/Goblin_Tinkerer] and Cubekon Tinkerer, with the Goblin Tinkerer offering a cheaper-but-ineffective means of it. The Cubekon Tinkerer, meanwhile, allows you to choose any modifier you would like to apply to your items, but the reforge process is significantly more expensive.

Post-Moon Lord modifiers

The Cubekon Tinkerer offers additional [link accessory | | Accessories.html] modifiers, which provide stronger boosts than the vanilla modifiers.

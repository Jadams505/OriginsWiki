Cursed Canister

[statblock Cursed_Canister]

The Cursed Canister is a type of [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link canister | | Canisters.html] used as [link ammunition] for [link launchers | | Launchers.html] and the [link Proximity Mine Launcher | | https://terraria.wiki.gg/wiki/Proximity_Mine_Launcher]. Upon landing on a tile or enemy, the Cursed Canister will explode into a lingering inferno, similar to [link Hellfire Grenades | | Hellfire_Grenade.html] and [link Bombs | | Hellfire_Bomb.html], that inflicts [link Cursed Inferno | | https://terraria.wiki.gg/wiki/Cursed_Inferno] to any enemies that cross it.

Ruby Ranger Armor

Ruby Ranger armor is a Hardmode armor set consisting of the Ruby Ranger Visor, Ruby Ranger Coat and Ruby Ranger Greaves. In order to craft the full set the player will need 24 Rubies, 9 Ichor and 72 Carburite.

A full set of Ruby Ranger armor provides the following benefits:

- +21 defense
- +10% ranged critical strike chance
- +10% movement speed
Set bonus
- Critical strikes inflict Bleeding on enemies
- Bleeding enemies receive 200% more damage

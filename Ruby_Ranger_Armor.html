Ruby Ranger Armor

[statblock Ruby_Armor]

Ruby Ranger armor is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link armor] set consisting of the Ruby Ranger Visor, Ruby Ranger Coat and Ruby Ranger Greaves. In order to craft the full set the player will need 24 [link Rubies | | https://terraria.wiki.gg/wiki/Ruby], 9 [link Ichor | | https://terraria.wiki.gg/wiki/Ichor] and 72 [link Carburite].

A full set of Ruby Ranger armor provides the following benefits:

- +21 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +10% [link ranged | | Ranged_Weapons.html] [link critical strike | | https://terraria.wiki.gg/wiki/Critical_hit] chance
- +10% movement speed
Set bonus
- Critical strikes inflict [link Bleeding | | https://terraria.wiki.gg/wiki/Bleeding] on enemies
- Bleeding enemies receive 200% more damage

Brainade

[statblock Brainade]

The Brainade is an [link explosive | | Demolitionist_Weapons.html] crafted using [link Crimtane Ore | | https://terraria.wiki.gg/wiki/Crimtane_Ore] and [link Grenades | | https://terraria.wiki.gg/wiki/Grenade]. They are also sold by the [link Demolitionist | | https://terraria.wiki.gg/wiki/Demolitionist] [link NPC | | NPCs.html] for [price 2|50] after selling him 81 [link Peat Moss] once the [link Brain of Cthulhu | | https://terraria.wiki.gg/wiki/Brain_of_Cthulhu] has been defeated. Brainades explode into 12-18 blood droplets that inflict [link Bleeding | | https://terraria.wiki.gg/wiki/Bleeding] for 3 seconds and [link Confused | | https://terraria.wiki.gg/wiki/Confused] for 2 seconds.

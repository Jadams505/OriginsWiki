Cicada 3301

[statblock Cicada_3301]

Cicada 3301 is a [link critter | | Critters.html] found in the [link Defiled Wastelands] when the player breaks any background objects found there. It can be caught with a [link Bug Net | | https://terraria.wiki.gg/wiki/Bug_Net] to be carried around in the [link inventory | | https://terraria.wiki.gg/wiki/Inventory] and released later, or crafted into animated decorative [link furniture] ([link Cicada 3301 Cage]). Once crafted, the Cicada 3301 cannot be released again, unless the furniture item is de-crafted via [link Shimmer | | https://terraria.wiki.gg/wiki/Shimmer]. Cicada 3301 can furthermore be used as [link bait | | Bait.html] for [link fishing | | https://terraria.wiki.gg/wiki/Fishing], with a bait power of 40%.

Nano Armor

[statblock Nano_Armor]

Nano armor is a post-Moon Lord armor set consisting of the Nano Mask, Nano Coat and Nano Pants. In order to craft the full set the player will need 60 Material-36 Bundles.

A full set of Nano armor provides the following benefits:

- +55 defense
- +15% magic damage
- +100 mana capacity
Set bonus
- Critical hits instead triple damage
- Every 6 mana regenerated heals 1 health
- Grants the 'Chameleon' ability by pressing the ability key

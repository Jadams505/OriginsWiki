Defiled Armor

[statblock Defiled_Armor]

Defiled armor is an [link armor] set consisting of the Defiled Helmet, Defiled Breastplate and Defiled Greaves. In order to craft the full set the player will need 60 [link Defiled Bars | | Defiled_Bar.html] (180 [link Lost Ore]) and 45 [link Undead Chunks | | Undead_Chunk.html].

Defiled armor pieces are interchangable with pieces from the [link Ancient Defiled armor] set.

A full set of Defiled armor provides the following benefits:

- +19 [link defense | | https://terraria.wiki.gg/wiki/Defense] - +18% [link critical strike | | https://terraria.wiki.gg/wiki/Critical_hit] chance - +6% damage Set bonus - +25% max [link life | | https://terraria.wiki.gg/wiki/Health]

Defiled Armor

{statblock { header:Defiled Armor, items:[ {image: DefiledSet} ] }, { header:Statistics, items:[ { label:Defense, value:19 (set) }, { label:Set-Bonus, value:"15% of damage is redirected to mana" }, { label:Rarity, value:Blue }, { label:Sell, value:1 Gold\, 80 Silver } ] } statblock}

Defiled Armor is an armor set that includes the Defiled Helmet, Defiled Breastplate, and Defiled Greaves. The total defense of the set is equivalent
to the vanilla alternatives already in the game. Each of the pieces provide a boost that revolves around the Magic-class as the armor set has a focus on
mana.

Wearing the full set makes it appear as if the player is losing and gaining chunks as Defiled particles fall off and orbit the player. Additionally, Defiled
chunks fly off of the player when damaged in a similar fashion to Defiled enemies.

Crafting a full set of Defiled armor requires 60 [link Defiled Bars] (180 [link Lost Ore]) and 45 [link Undead Chunks] in total.

Encrusted Ore

[statblock Encrusted_Ore]

Encrusted Ore is a type of [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link ore | | Ores.html] [link block | | Blocks.html] found in small veins [link Underground | | https://terraria.wiki.gg/wiki/Underground] or in larger quantities from the [link Eye of Cthulhu | | https://terraria.wiki.gg/wiki/Eye_of_Cthulhu] and [link World Cracker]. It is the Riven-equivalent to [link the Corruption's | | https://terraria.wiki.gg/wiki/The_Corruption] [link Demonite Ore | | https://terraria.wiki.gg/wiki/Demonite_Ore] or [link the Crimson's | | https://terraria.wiki.gg/wiki/The_Crimson] [link Crimtane Ore | | https://terraria.wiki.gg/wiki/Crimtane_Ore]. Encrusted Ore requires at least 55% [link pickaxe power | | https://terraria.wiki.gg/wiki/Pickaxe_power] to mine, and is exclusive to worlds with the [link Riven | | Riven_Hive.html] present.

Encrusted Ore emits a faint blue light in the dark.

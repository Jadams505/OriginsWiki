Flask of Black Bile

[statblock Bile_Flask]

The Flask of Black Bile is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link flask | | Flasks.html] that can be crafted using [link Bottled Water | | https://terraria.wiki.gg/wiki/Bottled_Water] and [link Black Bile]. When consumed, it allows all [link melee weapons] and [link whips] to inflict the [link Rasterized] debuff for 0.8 seconds. This effect will last for 20 minutes, but can be canceled anytime by right-clicking the icon. Melee weapons that fire [link projectiles | | https://terraria.wiki.gg/wiki/Projectiles] will also inflict the debuff on enemies hit by the projectiles (except for the [link Flairon | | https://terraria.wiki.gg/wiki/Flairon]'s bubbles).

The Flask of Black Bile cannot be used in conjunction with other flasks, and using it will remove any existing flask effects. Like all other flasks, the effect will not be canceled when the player [link dies | | https://terraria.wiki.gg/wiki/Death].

Cursed Crown

[statblock Cursed_Crown]

The Cursed Crown is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] that is dropped by [link King Slime | | https://terraria.wiki.gg/wiki/King_Slime] in [link Master Mode | | https://terraria.wiki.gg/wiki/Master_Mode]. When equipped, the player will dawn a blue, gelatinous appearance, can float in [link water | | https://terraria.wiki.gg/wiki/Water], are immune to [link fall damage | | https://terraria.wiki.gg/wiki/Fall_damage], and recieve twice as much damage from the [link On Fire! | | https://terraria.wiki.gg/wiki/On_Fire!] [link debuff | | https://terraria.wiki.gg/wiki/Debuffs]. Additionally, the player recieves the following alterations to their stats:

- -10% movement speed
- -40% acceleration/deceleration
- +75% jump speed
- +30% [link damage reduction | | https://terraria.wiki.gg/wiki/Damage_reduction]

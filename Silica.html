Silica

[statblock Silica]

Silica is a [link block | | Blocks.html] type found in a [link Desert | | https://terraria.wiki.gg/wiki/Desert] converted by the [link Riven | | Riven_Hive.html], or the [link Eutrophic Sea]. It is the Riven-equivalent to the [link sand blocks | | https://terraria.wiki.gg/wiki/Sand_Blocks].

Notes

- Any other type of sand becomes Silica when exposed to [link Riven Rain]. Silica gets further broken down into [link Silicon] when exposed to amebic gel, the Riven's [link water | | https://terraria.wiki.gg/wiki/Water] type.

Trivia

- [link Silica | | https://en.wikipedia.org/wiki/Silicon_dioxide] is a chemical compound of one silicon atom and two oxygen atoms. The reason it is prevalent with the Riven is because it is major portion of sand's composition and that Alkahest is notorious for breaking down anything that isn't a pure element.

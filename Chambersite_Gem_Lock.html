Chambersite Gem Lock

[statblock Chambersite_Gem_Lock]

The Chambersite Gem Lock is a special [link mechanism | | https://terraria.wiki.gg/wiki/Mechanisms] which acts as a container or altar for a [link Large Chambersite]. The gem lock will emit an activation signal a Large Chambersite is placed inside it and once again if it gets removed. This allows for a simple contraption that uses only one mechanism to keep something like a door open as long as the gem is inside the lock, because if it gets taken out it will immediately close.

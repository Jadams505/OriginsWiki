Celestial Stone Mask

[statblock Celestial_Mask]

The Celestial Stone Mask is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode], post-[link Golem | | https://terraria.wiki.gg/wiki/Golem] [link accessory | | Accessories.html] created by combining [link Celestial Stone | | https://terraria.wiki.gg/wiki/Celestial_Stone] and the [link Stone Mask]. Equipping it provides major boosts to the player's stats at the cost of their movement speed.

Stat boosts: - +15% melee speed - +10% [link generic damage] - +4% [link critical strike | | https://terraria.wiki.gg/wiki/Critical_hit] chance - +4 HP/s [link health regeneration | | https://terraria.wiki.gg/wiki/Health_regeneration] - +8 defense - +25% [link mining speed | | https://terraria.wiki.gg/wiki/Mining_speed] - +1 minion [link knockback | | https://terraria.wiki.gg/wiki/Knockback]

Stat reduction: - -10% movement speed - -33% jump speed

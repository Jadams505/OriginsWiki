Chlorophyte Visage

[statblock Chlorophyte_Visage]

The Chlorophyte Visage is a helmet armor piece to serve as the Demolitionist-equivalent for Chlorophyte armor. Equipping this as part of a full set does not change the set bonus provided and only offers basic combat stats.

The Chlorophyte Visage provides the following benefits:

- +5 defense
- +35% explosive blast radius
- -25% explosive fuse-time

Spirit Shard

[statblock Spirit_Shard]

The Spirit Shard is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] obtained when an [link Eccentric Stone] is nearby a [link Dark Lamia | | https://terraria.wiki.gg/wiki/Lamia]. While equipped, any [link artifact | | Artifacts.html] [link minions | | https://terraria.wiki.gg/wiki/Summon_weapons#Minions] that are slain will comeback as a ghost with half their health. This effect only works once per minion summoned by the player.

Mana Shielding

Mana Shielding is a [link mechanic | | https://terraria.wiki.gg/wiki/Game_mechanics] that is currently unique to [link Defiled | | Defiled_Wastelands.html]-themed items allowing damage to be redirected to [link mana | | https://terraria.wiki.gg/wiki/Mana] usage. The effect has a set amount of damage it converts to lost mana depending on the equipment using this mechanic. Every time any amount of incoming damage is converted to mana, the player is inflicted with the [link Mana Buffer] debuff for 3 seconds.

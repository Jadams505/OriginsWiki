Silicon

[statblock Silicon]

Silicon is a type of [link ore | | Ores.html] [link block | | Blocks.html] found in the [link Underground | | https://terraria.wiki.gg/wiki/Underground] and [link Cavern | | https://terraria.wiki.gg/wiki/Cavern] layers. It is the one of three ores that do not need to be turned into [link bars] in order to utilize as a crafting [link material | | Materials.html]. Silicon requires at least 35% [link pickaxe power | | https://terraria.wiki.gg/wiki/Pickaxe_power] or [link explosives | | Demolitionist_Weapons.html] to mine. It is used for some advanced crafting [link recipes] with the introduction of some more technology-based items.

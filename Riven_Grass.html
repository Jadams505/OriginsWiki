Riven Grass

[statblock Riven_Grass]

Riven Grass is a [link block | | Blocks.html] type found outside the [link Riven Hive]. It is the Riven-equivalent to [link Grass | | https://terraria.wiki.gg/wiki/Grasses], and can be converted to other variants with [link any thrown powder | | https://terraria.wiki.gg/wiki/Thrown_Powder]. Riven Grass can be placed and spread using [link Riven Seeds]. Given enough time, [link foliage | | Plants.html] will start to grow on top of Riven Grass.

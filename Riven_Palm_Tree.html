Riven Palm Tree

[statblock Riven_Palm_Tree]

Riven Palm Trees are a variant of [link palm trees | | https://terraria.wiki.gg/wiki/Trees#Palm_tree] grown on [link Silica] [link blocks] found at an [link Ocean | | https://terraria.wiki.gg/wiki/Ocean] or [link Oasis | | https://terraria.wiki.gg/wiki/Oasis] converted by the [link Riven | | Riven_Hive.html]. Unlike the vanilla equivalents, Riven Palm Trees do not have a dedicated wood item and drop [link Spug Flesh] upon destruction despite having to use an [link axe | | Axes.html] or [link chainsaw | | Chainsaws.html] to destroy it. Additionally, Riven Palm Trees have a 143/2000 (7.015%) chance to drop a [link Pawpaw] or [link Periven] in normal conditions. During the [link Halloween | | https://terraria.wiki.gg/wiki/Halloween] event, these chances are slightly reduced to a 34073/500000 (6.8146%). Uniquely from the [link Riven Tree], the drop rate for both is reduced to a 61089/1000000 (6.1089%) chance when the this tree is within an Ocean biome.

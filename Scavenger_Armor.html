Scavenger Armor

[statblock Scavenger_Armor]

Scavenger armor is an [link armor] set consisting of the Scavenger Goggles, Scavenger Body Armor and Scavenger Greaves. In order to craft the full set the player will need 42 [link Sanguinite Ore] and 66 [link Biocomponent 10].

A full set of Scavenger armor provides the following benefits:

- +10 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +10% [link explosive | | Demolitionist_Weapons.html] [link critical strike | | https://terraria.wiki.gg/wiki/Critical_hit] chance
- -25% [link self-damage | | Self_Damage.html]
- +15% explosive [link velocity | | https://terraria.wiki.gg/wiki/Velocity]
Set bonus
- All attacks inflict [link Impeding Shrapnel]
- 35% chance for additional [link Scrap] from any source

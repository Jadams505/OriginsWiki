Blast-Resistant Plate

[statblock Blast_Resistant_Plate]

The Blast-Resistant Plate is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link accessory | | Accessories.html] crafted using [link Adamantite Bars | | https://terraria.wiki.gg/wiki/Adamantite_Bar] or [link Titanium Bars | | https://terraria.wiki.gg/wiki/Titanium_Bar] with [link Obsidian | | https://terraria.wiki.gg/wiki/Obsidian]. It protects the player by reducing [link self-damage | | Self_Damage.html] by 15% and providing 3 defense. It is also a [link material] used to craft the [link Blast armor] set.

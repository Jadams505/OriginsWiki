Riven Armor

[statblock Riven_Armor]

Riven armor is an [link armor] set consisting of the Riven Mask, Riven Coat and Riven Pants. In order to craft the full set the player will need 60 [link Encrusted Bars | $default | Encrusted_Bar.html] (180 [link Encrusted Ore]) and 45 [link Riven Carapace].

Riven armor pieces are interchangable with pieces from the [link Ancient Riven armor] set.

A full set of Riven armor provides the following benefits:

- +10 [link defense | $default | https://terraria.wiki.gg/wiki/Defense]
- +18% attack speed Set bonus
- +10% [link Torn] severity

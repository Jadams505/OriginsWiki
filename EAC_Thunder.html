EAC Thunder

[statblock EAC_Thunder]

The EAC Thunder is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link gun | | Guns.html] crafted using mech components dropped by the mechanical bosses and some [link Valkyrum Bars | | Valkyrum_Bar.html]. It uses [link Power Cores | | Power_Core.html] for ammo and fires a giant stream of electric bolts that inflict Electrified to any enemies struck by it.

Its best [link modifier | | Modifiers.html] is Unreal.

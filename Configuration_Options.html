Configuration options

Origin's configuration options are changable settings provided by the mod in its in-game mod configuration menu. Configuration options can be set to modify visual or gameplay details as players see fit.

General options

- Set bonus double-tap: Requires a specified key binding (Down button by default) to be used twice in a short time-frame for set bonus abilities to trigger
- Animated Ravel transformation: Makes Ravel transformations flashier
- Extra-gooey Riven gores: Makes Riven enemies release a gooey gore that sticks to the player, NPCs, and other surfaces. Worm-like Riven enemies release a lot of these gores which may hinder performance.
- 24-hour time: Makes all clocks display time in a 24-hour format

Journal options

- Auto-open on unlock: Opens the Journal when inspecting an item for the first time
- Open Journal to: Changes the tab that the Journal always opens to (the Index, Quests, or Search tabs)

Vanilla Modifications

- Wooden item buffs: Whether or not vanilla/modded wooden items receive buffs from Origins

Other

- Universal Grass Merge: Forces grass to merge with other types of grass

Protean Potion

[statblock Protean_Potion]

The Protean Potion is a [link buff potion | | Buff_Potions.html] that can be crafted using [link Bottled Water | | https://terraria.wiki.gg/wiki/Bottled_Water], a [link Tearracuda] and any [link evil herb | | Recipes.html]. When consumed, it provides the Protean [link buff | | Buffs.html], which increases the player's [link stat share] by 50%. This effect will last for 4 minutes, but can be canceled anytime by right-clicking the icon.

Tips

- The Protean Potion has a unique [link item synergy | | Item_Synergies.html] involving the player filling all of their accessory slots with all of the class-specific [link emblems | | https://terraria.wiki.gg/wiki/Emblem] and then using a Protean Potion to amplify all of the damage for all weapons regardless of class. This playstyle will require the player to farm the [link Wall of Flesh | | https://terraria.wiki.gg/wiki/Wall_of_Flesh] at minimum of 8 times for all of the possible emblems (including the combo accessories) and [link fish | | https://terraria.wiki.gg/wiki/Fishing] in the [link Riven Hive] for an extended period in order to have enough potions.

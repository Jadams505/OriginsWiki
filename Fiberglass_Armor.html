Fiberglass Armor

[statblock Fiberglass_Armor]

Fiberglass armor is an [link armor] set consisting of the Fiberglass Helmet, Fiberglass Pauldrons and Fiberglass Boots. Each piece has a 1/4 (25%) chance of being dropped by the [link Fiberglass Weaver]. In order to obtain the full set the player will need to slay the Fiberglass Weaver at least 3 times.

A full set of FIberglass armor provides the following benefits:

- +16 [link defense | | https://terraria.wiki.gg/wiki/Defense] Set bonus - +4 damage

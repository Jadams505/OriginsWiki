Fiberglass Armor

[statblock Fiberglass_Armor]

Fiberglass armor is an [link armor] set consisting of the Fiberglass Helmet, Fiberglass Pauldrons and Fiberglass Boots. The [link Fiberglass Weaver] drops any of the one pieces of the set or two at a time in [link Expert Mode | | https://terraria.wiki.gg/wiki/Expert_Mode]. In order to obtain the full set the player will need to slay the Fiberglass Weaver at least 3 times.

A full set of FIberglass armor provides the following benefits:

- +16 [link defense | | https://terraria.wiki.gg/wiki/Defense]
Set bonus
- +4 damage [link generic damage]

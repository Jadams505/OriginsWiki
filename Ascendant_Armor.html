Ascendant Armor

Ascendant armor is a post-Moon Lord armor set consisting of the Ascendant Hood, Ascendant Crown, Ascendant Shirt and Ascendant Skirt. In order to craft the full set the player will need 60 Ethereal Ash for the Hood, and 55 Ethereal Ash for the Crown.

A full set of Ascendant armor provides the following benefits:

Ascendant Hood
- +60 defense
- +94% minion damage
- +30% whip attack speed
- +40 mana capacity
- +2 minion slots
- +15% movement speed
Set bonus
- All minions inflict Cursed Inferno to any enemies within 10 tiles of them
- All attacks inflict Celestial Flames
- Any enemies within 50 tiles of the player are permanently afflicted by On Fire!
- Gain health when inflicted with the Burning, Cursed Inferno, Shadowflame, Celestial Flames or On Fire! debuffs

Ascendant Crown
- +55 defense
- -45% mana consumption
- +3 minion slots
- +81% minion damage
- +40 mana capacity
- +15% movement speed
Set bonus
- Summons a phoenix minion that can be stacked using the player`s minion slots
- Gain health when inflicted with the Burning, Cursed Inferno, Shadowflame, Celestial Flames or On Fire! debuffs

Ascendant Armor

[statblock Ascendant_Armor]

Ascendant armor is a post-[link Moon Lord | | https://terraria.wiki.gg/wiki/Moon_Lord] [link armor] set consisting of the Ascendant Hood, Ascendant Crown, Ascendant Shirt and Ascendant Skirt. In order to craft the full set the player will need 60 [link Ethereal Ash] for the Hood, and 55 Ethereal Ash for the Crown.

A full set of Ascendant armor provides the following benefits:

Ascendant Hood
- +60 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +94% [link minion | | https://terraria.wiki.gg/wiki/Damage#Summon] damage
- +30% [link whip | | Whips.html] [link attack speed | | https://terraria.wiki.gg/wiki/Attack_speed]
- +40 [link mana | | https://terraria.wiki.gg/wiki/Mana] capacity
- +2 [link minion | | https://terraria.wiki.gg/wiki/Summon_weapons#Minions] slots
- +15% movement speed
Set bonus
- All minions inflict [link Cursed Inferno | | https://terraria.wiki.gg/wiki/Cursed_Inferno] to any enemies within 10 tiles of them
- All attacks inflict [link Celestial Flames]
- Any enemies within 50 tiles of the player are permanently afflicted by [link On Fire! | | https://terraria.wiki.gg/wiki/On_Fire!]
- Gain health when inflicted with the Burning, Cursed Inferno, Shadowflame, Celestial Flames or On Fire! debuffs

Ascendant Crown
- +55 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- -45% mana consumption
- +3 minion slots
- +81% minion damage
- +40 mana capacity
- +15% movement speed
Set bonus
- Summons a phoenix minion that can be stacked using the player`s minion slots
- Gain health when inflicted with the Burning, Cursed Inferno, Shadowflame, Celestial Flames or On Fire! debuffs

Edenwood

[statblock Eden_Wood]

Edenwood is a type of [link wood | | https://terraria.wiki.gg/wiki/Woods] [link block | | Blocks.html] and [link material | | Materials.html] dropped by [link Eden Trees] found at [link the Dawn]. Edenwood behaves the same as regular wood, but adopts a cherry-red coloration paired with eyes. It can only be obtained in [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] due to being solely accessible by the Dawn. Eden Trees can be planted anew by planting [link Acorns | | https://terraria.wiki.gg/wiki/Acorn] on [link Edengrass].

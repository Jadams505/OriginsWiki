Edenwood

[statblock Eden_Wood]

Edenwood is a type of [link wood | | https://terraria.wiki.gg/wiki/Woods] [link block | | Blocks.html] and [link material | | Materials.html] droped by on [link Eden Trees] found at [link the Dawn]. Eden Trees require an [link axe | | Axes.html] or [link chainsaw | | Chainsaws.html] to destroy and drop [link Edenwood], which can be used to craft [link Edenwood armor]. Eden Trees currently do not have a [link fruit | | https://terraria.wiki.gg/wiki/Fruits] drop obtained by [link shaking | | https://terraria.wiki.gg/wiki/Trees#Shaking] them.

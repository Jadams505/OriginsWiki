Soul Scythe

[statblock Soul_Scythe]

The Soul Scythe is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] that has a 1/2 (50%) chance of being dropped by the [link Treasure Bag | | https://terraria.wiki.gg/wiki/Treasure_Bag] dropped by the [link Eater of Worlds | | https://terraria.wiki.gg/wiki/Eater_of_Worlds] in [link Expert Mode | | https://terraria.wiki.gg/wiki/Expert_Mode]. When equipped, ghastly blades will be launched at nearby enemies. This item shares its loot pool with the [link Worm Scarf | | https://terraria.wiki.gg/wiki/Worm_Scarf].

Trivia

- The Soul Scythe is a nod to the removed item from the vanilla game [link with the same name | | https://terraria.wiki.gg/wiki/Soul_Scythe].

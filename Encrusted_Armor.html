Encrusted Armor

[statblock Encrusted_Armor]

Encrusted armor is an [link armor] set consisting of the Encrusted Mask, Encrusted Coat and Encrusted Pants. In order to craft the full set the player will need 42 [link Encrusted Ore] and 66 [link Bud Barnacle].

A full set of Encrusted armor provides the following benefits:

- +10 [link defense | | https://terraria.wiki.gg/wiki/Defense] - +10% [link minion damage | | Summon_Weapons.html] - +1 [link minion | | https://terraria.wiki.gg/wiki/Summon_weapons#Minions] slot - +15% jump speed Set bonus - +30% minion damage when above half [link health | | https://terraria.wiki.gg/wiki/Health]

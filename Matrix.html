Matrix

[statblock Matrix]

The Matrix is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link launcher | | Launchers.html] that can be crafted using [link NE-8 | | NE_8.html] and [link Sanguinite Bars | | Sanguinite_Bar.html]. The Matrix converts its [link ammunition] into 3-5 hovering nodes, similarly to the [link Bombardment], that connect to each other through a laser beam. When an enemy has entered the shape crated by the nodes, the area within the shape will explode.

Its best [link modifier | | Modifiers.html] is Guileless.

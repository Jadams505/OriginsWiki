Ichor Canister

[statblock Ichor_Canister]

The Ichor Canister is a type of [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link canister | | Canisters.html] used as [link ammunition] for [link launchers | | Launchers.html] and the [link Proximity Mine Launcher | | https://terraria.wiki.gg/wiki/Proximity_Mine_Launcher]. Upon landing on a tile or enemy, the Ichor Canister will explode and have a chance to inflict the [link Ichor debuff | | https://terraria.wiki.gg/wiki/Ichor_(debuff)] to any enemies within its blast radius. Directly hit enemies will instead be guaranteed to be receive a longer lasting Ichor debuff.

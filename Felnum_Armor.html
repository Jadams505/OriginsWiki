Felnum Armor

[statblock Felnum_Armor]

Felnum armor is an [link armor] set consisting of the Felnum Helmet, Felnum Breastplate and Felnum Greaves. In order to craft the full set the player will need 60 [link Felnum Bars | | Felnum_Bar.html] (180 [link Felnum Ore]).

A full set of Felnum armor provides the following benefits:

- +16 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +6% damage
- 10% movement speed
Set bonus
- Movement speed builds up the static charge of the armor which increases the damage dealt on the player's next strike

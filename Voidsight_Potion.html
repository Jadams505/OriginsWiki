Voidsight Potion

[statblock Voidsight_Potion]

The Voidsight Potion is a [link buff potion | | Buff_Potions.html] that can be crafted using a [link Fireblossom | | https://terraria.wiki.gg/wiki/Fireblossom], [link Night Owl Potion | | https://terraria.wiki.gg/wiki/Night_Owl_Potion] and a [link Duskarp]. When consumed, it provides the Voidsight [link buff | | Buffs.html], which increases the apparent light level provided by all light sources on screen and causes the player to emit a lot of light themselves. This effect will last for 6 minutes, but can be canceled anytime by right-clicking the icon.

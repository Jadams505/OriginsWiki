Bomb Handling Device

[statblock Bomb_Handling_Device]

The Bomb Handling Device is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] found in [link Locked Gold Chests | | https://terraria.wiki.gg/wiki/Gold_Chest#Dungeon] stowed away in the [link Dungeon | | https://terraria.wiki.gg/wiki/Dungeon]. The player will be granted a 25% boost to the [link velocity | | https://terraria.wiki.gg/wiki/Velocity] of all [link thrown | | https://terraria.wiki.gg/wiki/Throwing_weapons] [link explosives | | Demolitionist_Weapons.html].

Nova Armor

[statblock Nova_Armor]

Nova armor is a post-[link Moon Lord | | https://terraria.wiki.gg/wiki/Moon_Lord] [link armor] set consisting of the Nova Faceshield, Nova Vest and Nova Leggings. In order to craft the full set the player will need 36 [link Luminite Bars | | https://terraria.wiki.gg/wiki/Luminite] (144 [link Luminite | | https://terraria.wiki.gg/wiki/Luminite_Ore]) and 45 [link Nova Fragments | | Nova_Fragment.html].

A full set of Nova armor provides the following benefits:

- +48 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +12% [link explosive | | Demolitionist_Weapons.html] damage
- +12% explosive [link critical strike | | https://terraria.wiki.gg/wiki/Critical_hit] chance
- +50% explosive [link velocity | | https://terraria.wiki.gg/wiki/Velocity]
- -60% [link self-damage | | Self_Damage.html]
- +15% movement speed
Set bonus
- +100% explosive blast radius
- 50% chance to duplicate explosive projectiles
- Explosive projectiles home to enemies

Eyndum Armor

Eyndum armor is a post-Moon Lord armor set. It has melee, ranged, magic, summoner and explosive set versions, depending on the equipped headpiece. In order to obtain a full, individual set the player will need to complete the Test of the Ancients event at least 3 times, however, the event will need to be completed at least 7 times for all pieces.

A full set of Eyndum armor provides the following benefits:

Eyndum Helmet
- +44 defense
- +25% melee weapon size

Eyndum Tarnhelm
- +41 defense
- +40% ammo conservation

Eyndum Hood
- +38 defense
- -40% mana consumption

Eyndum Crown
- +35 defense
- +3 minion slots

Eyndum Globe
- +32 defense
- +50% explosive velocity

Set bonus
- Opens up a slot to install Eyndum Cores

Eyndum Armor

[statblock Eyndum_Armor]

Eyndum armor is a post-[link Moon Lord | | https://terraria.wiki.gg/wiki/Moon_Lord] [link armor] set. It has melee, ranged, magic, summoner and explosive set versions, depending on the equipped headpiece. In order to obtain a full, individual set the player will need to complete the [link Test of the Ancients] [link event | | Events.html] at least 3 times, however, the event will need to be completed at least 7 times for all pieces.

A full set of Eyndum armor provides the following benefits:

Eyndum Helmet
- +44 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +25% [link melee | | Melee_Weapons.html] weapon size

Eyndum Tarnhelm
- +41 defense
- +40% [link ammo conservation | | https://terraria.wiki.gg/wiki/Ammo_conservation]

Eyndum Hood
- +38 defense
- -40% [link mana | | https://terraria.wiki.gg/wiki/Mana] consumption

Eyndum Crown
- +35 defense
- +3 [link minion | | https://terraria.wiki.gg/wiki/Summon_weapons#Minions] slots

Eyndum Globe
- +32 defense
- +50% [link explosive | | Demolitionist_Weapons.html] [link velocity | | https://terraria.wiki.gg/wiki/Velocity]

Set bonus
- Opens up an slot to install [link Eyndum Cores]

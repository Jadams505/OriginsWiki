Tree Sap

[statblock Tree_Sap]

Tree Sap is [link material | | Materials.html] that is harvested by [link trees | | https://terraria.wiki.gg/wiki/Trees] from either chopping them down or [link shaking | | https://terraria.wiki.gg/wiki/Trees#Shaking] them. It is used to create [link Rubber], [link Adhesive Wrap] and [link Lesser Healing Potions | | https://terraria.wiki.gg/wiki/Lesser_Healing_Potion].

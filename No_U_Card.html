No U Card

[statblock No_U_Card]

The No U Card is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link immunity | | https://terraria.wiki.gg/wiki/Immunity_accessories] [link accessory | | Accessories.html] crafted using an [link Ankh Charm | | https://terraria.wiki.gg/wiki/Ankh_Charm] and a [link Return To Sender]. It retaliates against enemies that have struck the player, returning 50% of damage recieved and applies any [link debuffs | | https://terraria.wiki.gg/wiki/Debuffs] that would've applied to the player. Additionally, it grants the player immunity to the following debuffs:

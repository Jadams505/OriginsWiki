Scavenger Bag

[statblock Scavenger_Bag]

The Scavenger Bag is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] post-[link Plantera | | https://terraria.wiki.gg/wiki/Plantera] [link accessory | | Accessories.html] combining the effects of the [link Papyrus Scarab | | https://terraria.wiki.gg/wiki/Papyrus_Scarab] and [link Keepsake Remains]. While equipped, the player's [link armor penetration | | https://terraria.wiki.gg/wiki/Defense#Armor_penetration] is increased by 5, max number of [link minions | | https://terraria.wiki.gg/wiki/Summon_weapons#Minions] by 1, [link summon damage | | https://terraria.wiki.gg/wiki/Damage#Summon] by 15% and the [link knockback | | https://terraria.wiki.gg/wiki/Knockback] of minions by 2. Additionally, all of the player's attacks inflict [link Torn].

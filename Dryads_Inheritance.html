Dryad's Inheritance

[statblock Dryads_Inheritance]

The Dryad's Inheritance is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode], [link Expert Mode | | https://terraria.wiki.gg/wiki/Expert_Mode]-exclusive, post-[link Plantera | | https://terraria.wiki.gg/wiki/Plantera] [link accessory | | Accessories.html] combining the [link Spore Sac | | https://terraria.wiki.gg/wiki/Spore_Sac] and the [link Last Descendant]. While equipped, this increases the player's damage, movement speed, and length of [link invincibility | | https://terraria.wiki.gg/wiki/Invincibility_frame] after taking damage. Additionally, it casts an aura around the player that deals constant damage and the [link Poisoned | | https://terraria.wiki.gg/wiki/Poisoned] [link debuff | | Debuffs.html] to any enemies within. The aura deals even more damage against [link world evil | | https://terraria.wiki.gg/wiki/Evil_biomes] enemies.

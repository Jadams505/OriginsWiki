Ornament of Metamorphosis

[statblock Ornament_of_Metamorphosis]

The Ornament of Metamorphosis is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link accessory | | Accessories.html] combining the [link Moon Charm | | https://terraria.wiki.gg/wiki/Moon_Charm], [link Neptune's Shell | | https://terraria.wiki.gg/wiki/Neptune%27s_Shell] and a [link Strange Feather]. Equipping it will turn the player into a Merfolk when entering [link water | | https://terraria.wiki.gg/wiki/Water] or [link honey | | https://terraria.wiki.gg/wiki/Honey] (at any time), Werewolf at [link night | | https://terraria.wiki.gg/wiki/Day_and_night_cycle] or a [link Half-Harpy | | Strange_Feather.html] when in [link Space | | https://terraria.wiki.gg/wiki/Space]. The Merfolk form replaces the Werewolf form if entering water at night and the Half-Harpy form replaces either regardless. The Ornament takes the place of the [link Moon Shell | | https://terraria.wiki.gg/wiki/Moon_Shell], but the both are still craftable for the the player, however the [link Celestial Shell | | https://terraria.wiki.gg/wiki/Celestial_Shell]'s crafting recipe now requires the Ornament instead and is not craftable otherwise.

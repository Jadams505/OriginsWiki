Amoeba Buggy

[statblock Amoeba_Buggy]

The Amoeba Buggy is a [link critter | | Critters.html] found in the [link Riven Hive] when the player breaks any background objects found there or when destroying a [link Barnacle Mound]. It can be caught with a [link Bug Net | | https://terraria.wiki.gg/wiki/Bug_Net] to be carried around in the [link inventory | | https://terraria.wiki.gg/wiki/Inventory] and released later, or crafted into animated decorative [link furniture] ([link Amoeba Buggy Cage]). Once crafted, the Amoeba Buggy cannot be released again, unless the furniture item is de-crafted via [link Shimmer | | https://terraria.wiki.gg/wiki/Shimmer]. The Amoeba Buggy can furthermore be used as [link bait | | Bait.html] for [link fishing | | https://terraria.wiki.gg/wiki/Fishing], with a bait power of 40%.

Riven Splitter

[statblock Riven_Splitter]

The Riven Splitter is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link harpoon gun | | Harpoon_Guns.html]. It is one of the possible drops from destroying a [link Riven Lesion] or [link Gel Blister]. It is normally obtained alongside 99 [link Harpoons | | Harpoon.html] to be used from the harpoon gun.

Its best [link modifier | | Modifiers.html] is Godly or Demonic.

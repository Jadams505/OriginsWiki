Endowood

[statblock Endowood]

Endowood is a type of [link wood | | https://terraria.wiki.gg/wiki/Woods] [link block | | Blocks.html] and [link material | | Materials.html] dropped by [link Petrified Trees] found in the [link Defiled Wastelands]. Endowood behaves the same as regular wood, but adopts a black-purple coloration with white stripes. Petrified Trees can be planted anew by planting [link Acorns | | https://terraria.wiki.gg/wiki/Acorn] on [link Defiled Grass].

Sonic Radar

[statblock Sonic_Radar]

Sonic Radar is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] crafted using several materials found underground. It specifically allows the player to see valuables, enemies, and traps from a 20 tile radius.

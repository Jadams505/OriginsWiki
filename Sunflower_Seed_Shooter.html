Sunflower Seed Shooter

[statblock Sunflower_Seed_Shooter]

The Sunflower Seed Shooter is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link gun | | Guns.html] that can be crafted using [link Sunflowers | | https://terraria.wiki.gg/wiki/Sunflower] and [link Wood | | https://terraria.wiki.gg/wiki/Woods]. It is meant to be the starter gun for [link rangers | | Ranged_Weapons.html] with its easy crafting recipe and inexpensive [link ammunition] source of Sunflowers. It has a modest [link fire rate | | https://terraria.wiki.gg/wiki/Attack_speed] and a 33% chance [link not to consume ammo | | https://terraria.wiki.gg/wiki/Ammo_conservation].

Its best [link modifier | | Modifiers.html] is Demonic or Deadly.

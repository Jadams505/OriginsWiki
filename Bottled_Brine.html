Bottled Brine

[statblock Bottled_Brine]

Bottled Brine is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link material | | Materials.html] that is brewed using [link Bottles | | https://terraria.wiki.gg/wiki/Bottle], [link Stingers | | https://terraria.wiki.gg/wiki/Stinger] and [link Magic Brine Droppers]. It cannot be directly obtained like [link water | | https://terraria.wiki.gg/wiki/Water] and must be crafted. It is used to make most [link Brine | | Brine_Pool.html]-themed items and for some brewing.

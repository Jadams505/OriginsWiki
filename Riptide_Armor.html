Riptide Armor

[statblock Riptide_Armor]

Riptide armor is an [link armor] set consisting of the Riptide Helm, Riptide Breastplate and Riptide Greaves. In order to craft the full set the player will need a [link Diving Helmet | | https://terraria.wiki.gg/wiki/Diving_Helmet], 39 [link Shell Piles | | https://terraria.wiki.gg/wiki/Shell_Pile], any [link Jellyfish | | https://terraria.wiki.gg/wiki/Jellyfish_(bait)] and a [link Flipper | | https://terraria.wiki.gg/wiki/Flipper].

A full set of Riptide armor provides the following benefits:

- +13 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +10% [link magic | | Magic_Weapons.html]
- +6 seconds of [link breath | | https://terraria.wiki.gg/wiki/Breath_meter]
- Emits a small aura of light
- +60 [link mana | | https://terraria.wiki.gg/wiki/Mana]
- Grants the ability to [link swim | | https://terraria.wiki.gg/wiki/Swimming]
- +10% movement speed when submerged
Set bonus
- +5% magic damage when submerged or in [link rain | | https://terraria.wiki.gg/wiki/Rain]
- Grants the ability to dash, releasing a damaging tidal wave projectile

Furniture

If you are looking for the official page of vanilla furniture, click here.

Furniture is a class of [link items | | https://terraria.wiki.gg/wiki/Items] that can be [link placed | | https://terraria.wiki.gg/wiki/Placement] on [link blocks], or hung from them, usually to become background objects that characters can move over and through. Some are functional, such as [link crafting stations], [link Chests], [link Beds | | https://terraria.wiki.gg/wiki/Beds], [link mechanisms | | https://terraria.wiki.gg/wiki/Mechanisms], and [link light sources] (among others), while many are mainly decorative. Furniture is acquired through crafting, by collecting furniture items that appear naturally in the world, dropped by [link Pirates | | https://terraria.wiki.gg/wiki/Pirate_Invasion], or purchasing from [link NPCs | | https://terraria.wiki.gg/wiki/NPCs]. Most placed furniture can be freed with a [link pickaxe | | Pickaxes.html].

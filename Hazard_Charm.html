Hazard Charm

[statblock Hazard_Charm]

The Hazard Charm is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] crafted with the [link Lava Charm | | https://terraria.wiki.gg/wiki/Lava_Charm], [link Bomb Charm] and [link Trap Charm]. It protects the player from most traps underground specifically providing a 15% reduction in [link self-damage | | Self_Damage.html], immunity to [link Poisoned | | https://terraria.wiki.gg/wiki/Poisoned] and half of all damage from [link traps | | https://terraria.wiki.gg/wiki/Traps] [link Underground | | https://terraria.wiki.gg/wiki/Underground], and a 7 second immunity to [link lava | | https://terraria.wiki.gg/wiki/Lava].

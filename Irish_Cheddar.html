Irish Cheddar

[statblock Irish_Cheddar]

Irish Cheddar is a [link food] item that has a 1/3 (33%) chance to drop from [link Spore Skeletons | | https://terraria.wiki.gg/wiki/Spore_Skeleton]. When consumed, it provides the Irish Cheddar [link buff | | Buffs.html], which has the effect of [link Well Fed | | https://terraria.wiki.gg/wiki/Well_Fed] and an 8% increase in their [link critical strike | | https://terraria.wiki.gg/wiki/Critical_hit] chance and lasts for 12 minutes.

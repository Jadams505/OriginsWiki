Treasure Bag

[statblock Treasure_Bag]

If you are looking for the official page of vanilla treasure bags, click [link here | | https://terraria.wiki.gg/wiki/Treasure_Bag].

Treasure Bagsb> are [link grab bag | | Grab_Bags.html]-like items obtained in [link Expert Mode | | https://terraria.wiki.gg/wiki/Expert_Mode] and [link Master Mode | | https://terraria.wiki.gg/wiki/Master_Mode] as a reward for defeating [link bosses]. The contents of Treasure Bags depend on the boss it comes from, and are opened by right-clicking while in the player's [link inventory | | https://terraria.wiki.gg/wiki/Inventory] or left-clicking while selected in the player's [link hotbar | | https://terraria.wiki.gg/wiki/Inventory#Hotbar_(10_slots)]. In [link multiplayer | | https://terraria.wiki.gg/wiki/Multiplayer], each player that deals damage to the boss receives one Treasure Bag upon defeating it.

Each Treasure Bag contains the boss's usual drops from [link Classic Mode | | https://terraria.wiki.gg/wiki/Classic_Mode], along with one Expert Mode-exclusive item. Treasure Bags dropped from [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] bosses additionally have a 1/16 (6.25%) chance to contain one random set of [link developer items | | https://terraria.wiki.gg/wiki/Developer_items].

Origins adds Master Mode-exclusive items, which also appear in treasure bags and share a drop pool with the Expert Mode-exclusive items that already exist.

Nova Monolith

[statblock Nova_Monolith]

The Nova Monolith is a functional [link furniture] item crafted using [link Nova Fragments | | Nova_Fragment.html] at an [link Ancient Manipulator | | https://terraria.wiki.gg/wiki/Ancient_Manipulator]. When placed, it can be toggled with the Open/Activate button or from [link wire | | https://terraria.wiki.gg/wiki/Wire] like other [link mechanisms | | https://terraria.wiki.gg/wiki/Mechanisms]. When activated, the screen and background effects for the [link Nova Pillar] and its preceding [link lunar event | | https://terraria.wiki.gg/wiki/Lunar_Events]. Like the other [link monoliths | | https://terraria.wiki.gg/wiki/Monoliths], it can be equipped as a [link vanity accessory | | https://terraria.wiki.gg/wiki/Vanity_Accessories] to enable the effect globally for the player.

Sniper's Mark

[statblock Snipers_Mark]

Sniper's Mark is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link accessory | | Accessories.html] combining the [link Ranger Emblem | | https://terraria.wiki.gg/wiki/Ranger_Emblem] and [link Ruby Reticle]. While equipped, [link ranged damage | | Ranged_Weapons.html] is increased by 15% and the player's [link critical strike | | https://terraria.wiki.gg/wiki/Critical_hit] chance is increased by 15% of the current held weapon's base damage.

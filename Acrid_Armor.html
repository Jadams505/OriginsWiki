Acrid Armor

[statblock Acrid_Armor]

Acrid armor is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link armor] set. It has melee, ranged, magic, summoner and explosive set versions, depending on the equipped headpiece. In order to craft the full individual set the player will need 59 [link Eitrite Bars | | Eitrite_Bar.html] and 60 [link Rubber], however, the player will need 111 Eitrite Bars and 100 Rubber total in order to craft all pieces of the set.

A full set of Acrid armor provides the following benefits:

Acrid Dome
- +34 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +10% [link explosive | | Demolitionist_Weapons.html] [link critical strike | | https://terraria.wiki.gg/wiki/Critical_hit] chance (+8% more while submerged)
- +10% explosive [link velocity | | https://terraria.wiki.gg/wiki/Velocity] (+8% more while submerged)
- Increased visibility while submerged
- Emit an aura of light
- +2 HP/s [link life regeneration | | https://terraria.wiki.gg/wiki/Health_regeneration]
- Grants the ability to [link swim | | https://terraria.wiki.gg/wiki/Swimming]

Acrid Headgear
- +36 defense
- +10% [link magic | | Magic_Weapons.html] damage (+8% more while submerged)
- -10% [link mana | | https://terraria.wiki.gg/wiki/Mana] cost (-10% more while submerged)
- Increased visibility while submerged
- Emit an aura of light
- +2 HP/s life regeneration
- Grants the ability to swim

Acrid Helm
- +50 defense
- +10% [link melee | | Melee_Weapons.html] damage (+8% more while submerged)
- +10% melee [link attack speed | | https://terraria.wiki.gg/wiki/Attack_speed] (+8% more while submerged)
- Increased visibility while submerged
- Emit an aura of light
- +2 HP/s life regeneration
- Grants the ability to swim

Acrid Mask
- +33 defense
- +10% [link summon | | Summoner_Weapons.html] damage (+8% more while submerged)
- +1 [link minion | | https://terraria.wiki.gg/wiki/Summon_weapons#Minions] slot (+1 more while submerged)
- Increased visibility while submerged
- Emit an aura of light
- +2 HP/s life regeneration
- Grants the ability to swim

Acrid Visor
- +41 defense
- +20% [link ranged | | Ranged_Weapons.html] attack speed (+16% more while submerged)
- Increased visibility while submerged
- Emit an aura of light
- +2 HP/s life regeneration
- Grants the ability to swim

Set bonus
- All attacks inflict [link Toxic Shock]
- Immunity to [link Acid Venom | | https://terraria.wiki.gg/wiki/Acid_Venom], [link Poisoned | | https://terraria.wiki.gg/wiki/Poisoned] and [link Toxic Shock]
- +6 seconds of [link breath | | https://terraria.wiki.gg/wiki/Breath_meter]

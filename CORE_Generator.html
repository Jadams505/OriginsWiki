Core Generator

[statblock CORE_Generator]

The CORE Generator is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link accessory | | Accessories.html] that combines the [link Last Descendant] and [link Missile Armcannon]. Swinging while this is equipped will fire any [link rockets | | https://terraria.wiki.gg/wiki/Rockets] in the player's inventory from their chest. Additionally, equipping the Armcannon provides these benefits:

- +30% [link explosive | | Demolitionist_Weapons.html] [link velocity | | https://terraria.wiki.gg/wiki/Velocity]
- +10% explosive [link attack speed | | https://terraria.wiki.gg/wiki/Attack_speed]
- +15% explosive blast radius
- +50% explosive [link knockback | | knockback | | https://terraria.wiki.gg/wiki/Knockback]
- [link Auto-use | | https://terraria.wiki.gg/wiki/Autoswing] on all explosive weapons - +20% [link generic damage], +50% movement speed and doubled length of [link invincibility | | https://terraria.wiki.gg/wiki/Invincibility_frame] after taking damage
- Rapidly release [link canisters | | Canister.html] from your [link inventory | | https://terraria.wiki.gg/wiki/Inventory] and stars will fall from the sky after taking damage

Bug Trapper

[statblock Bug_Trapper]

The Bug Trapper is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] combining a [link Shackle | | https://terraria.wiki.gg/wiki/Shackle] with the [link Primordial Soup]. When equipped, the player's [link life regeneration | | https://terraria.wiki.gg/wiki/Health_regeneration] and [link defense | | https://terraria.wiki.gg/wiki/Defense] will be increased depending on the current severity of [link Torn] being applied to them. Visually, the Bug Trapper will generate a swam of flying bugs around the player when equipped.

Defiled Seeds

[statblock Defiled_Grass_Seeds]

Defiled Seeds are [link seeds] which can be planted on [link Dirt Blocks | | https://terraria.wiki.gg/wiki/Dirt_Block] or [link Mud Blocks | | https://terraria.wiki.gg/wiki/Mud_Block] to create [link Defiled Grass] and [link Defiled Jungle Grass] respectively, which will then [link spread | | https://terraria.wiki.gg/wiki/Biome_spread] the [link Defiled | | Defiled_Wastelands.html] in all directions. They drop from the [link Eye of Cthulhu | | https://terraria.wiki.gg/wiki/Eye_of_Cthulhu] and can be purchased from the [link Dryad | | https://terraria.wiki.gg/wiki/Dryad] [link NPC | | NPCs.html] for [price 5|0] each during a [link Blood Moon | | https://terraria.wiki.gg/wiki/Blood_Moon] in a world with Defiled. In a world with any other [link world evil | | https://terraria.wiki.gg/wiki/Evil_biomes], Defiled Seeds can be purchased from the Dryad if she is spoken to in a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link Graveyard | | https://terraria.wiki.gg/wiki/Graveyard].

Spug Flesh

[statblock Riven_Flesh]

Spug Flesh is a [link block | | Blocks.html] type found in the [link Riven_Hive]. It is the Riven-equivalent to [link Stone | | https://terraria.wiki.gg/wiki/Stone_Block], and can be converted to other variants with [link any thrown powder | | https://terraria.wiki.gg/wiki/Thrown_Powder]. Spug Flesh requires at least 65% [link pickaxe power | | https://terraria.wiki.gg/wiki/Pickaxe_power] or [link explosives | | Demolitionist_Weapons.html] to mine.

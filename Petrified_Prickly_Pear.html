Petrified Prickly Pear

[statblock Petrified_Prickly_Pear]

The Petrified Prickly Pear is a [link food] item that has a 3/100 (3%) chance to drop from [link Petrified Trees | | Petrified_Tree.html] when [link shaken | | https://terraria.wiki.gg/wiki/Shaking]. When consumed, it provides the [link Well Fed | | https://terraria.wiki.gg/wiki/Well_Fed] [link buff | | Buffs.html] and 25% increased [link mana shielding], which lasts for 5 minutes.

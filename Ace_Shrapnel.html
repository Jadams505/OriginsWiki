Ace Shrapnel

[statblock Ace_Shrapnel]

The Ace Shrapnel is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link launcher | | Launchers.html] that is crafted using [link NE-8 | | NE_8.html] and [link Scrap], rendering it currently unobtainable through conventional means. The Ace Shrapnel fires a constant stream of shrapnel somewhat similar to the [link Flamethrower | | https://terraria.wiki.gg/wiki/Flamethrower] that inflict [link Impeding Shrapnel] for 8 seconds. It requires [link Scrap] as [link ammunition] to fire this weapon.

The best [link modifier | | Modifiers.html] is Heavy-Duty for its damage boost, so that enemies afflicted by Impeding Shrapnel can spread the effect faster.

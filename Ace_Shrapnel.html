Ace Shrapnel

[statblock Ace_Shrapnel]

The Ace Shrapnel is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link launcher | | Launchers.html] that is obtained from the [link Broken Record] or crafted using [link NE-8 | | NE_8.html] and [link Scrap]. The Ace Shrapnel fires a constant stream of frenzied shrapnel, in a manner somewhat similar to the [link Flamethrower | | https://terraria.wiki.gg/wiki/Flamethrower], that inflict [link Impeding Shrapnel] for 5 seconds. It requires [link Scrap] as [link ammunition] to fire this weapon.

The best [link modifier | | Modifiers.html] is Heavy-Duty for its damage boost, such that enemies afflicted by Impeding Shrapnel can spread the effect faster.

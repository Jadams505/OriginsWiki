Abysswalker Armor

[statblock Abysswalker_Armor]

Abysswalker armor is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link armor] set consisting of the Abysswalker's Hood, Abysswalker's Cloak and Abysswalker's Greaves. In order to craft the full set the player will need 12 [link Souls of Night | | https://terraria.wiki.gg/wiki/Soul_of_Night], 48 of any [link evil herb | | Recipes.html], 78 of any [link evil tissue | | Recipes.html], 90 of any [link evil boss material | | Recipes.html] and 60 of any [link evil substance | | Recipes.html]. Wearing all pieces of the set displays a UI element next to the player's armor slots where they can select from 12 unique abilities. These abilities require a certain amount of [link assimilation] from their respective [link world-evils | | https://terraria.wiki.gg/wiki/Evil_biomes] in order to be unlocked, pushing the player further into their risks.

A full set of Abysswalker armor provides the following benefits:

- +24 defense
- +20 max [link life | | https://terraria.wiki.gg/wiki/Health]
- +2 HP/s [link life regeneration | | https://terraria.wiki.gg/wiki/Health_regeneration]
- +15% movement speed
Set bonus
- +50% max assimilation
- Gain different abilities depending on the assimilation amount for each world evil

Abilities

Threshold 1

Rot
Requires 30% [link Corruption | | https://terraria.wiki.gg/wiki/The_Corruption] Assimilation
- All attacks have a chance to inflict [link Cursed Inferno | | https://terraria.wiki.gg/wiki/Cursed_Inferno] and [link Shadowflame | | https://terraria.wiki.gg/wiki/Shadowflame] simultaneously

Bleed
Requires 24% [link Crimson | | https://terraria.wiki.gg/wiki/The_Crimson] Assimilation
- All attacks have a chance to inflict [link Bleeding | | https://terraria.wiki.gg/wiki/Bleeding] or [link Ichor | | https://terraria.wiki.gg/wiki/Ichor_(debuff)]. Always applies Ichor if using a weapon or accessory that inflicts Bleeding

DEPRESS
Requires 15% [link Defiled | | Defiled_Wastelands.html] Assimilation
- All attacks inflict [link Weak | | https://terraria.wiki.gg/wiki/Weak] and a prompt [link Rasterized] simultaneously. Doubled Rasterized duration if under the effect of a [link Bile Flask]

Dissolve
Requires 10% [link Riven | | Riven_Hive.html] Assimilation
- All attacks have a chance to inflict [link Torn]. Torn severity increased by 7% if using a weapon or accessory that already inflicts Torn

Threshold 2

Natural Hunter
Requires 50% [link Corruption | | https://terraria.wiki.gg/wiki/The_Corruption] Assimilation
- A clinger tethered to you pulls enemies toward you, dealing damage

Goreweaver
Requires 42% [link Crimson | | https://terraria.wiki.gg/wiki/The_Crimson] Assimilation
- Release a gooey residue as you move that increases the damage of friendly projectiles that pass through it by 35%

EVOLVE
Requires 45% [link Defiled | | Defiled_Wastelands.html] Assimilation
- A Defiled Monolith will follow you and consume your [link mana | | https://terraria.wiki.gg/wiki/Mana] to ward off enemies or deflect 45% of damage received [link to mana | | Mana_Shielding.html]

Symbiosis
Requires 25% [link Riven | | Riven_Hive.html] Assimilation
- Bud barnacles grow on your back providing 15% increased [link damage reduction], an increase of your max [link minions | | https://terraria.wiki.gg/wiki/Summon_weapons#Minions] by 2, and summons any Riven-related
summon to fight for you at no cost

Threshold 3

Resilience
Requires 70% [link Corruption | | https://terraria.wiki.gg/wiki/The_Corruption] Assimilation
- Defense increased by 6 and damage reduction by 10%

Mesmer
Requires 66% [link Crimson | | https://terraria.wiki.gg/wiki/The_Crimson] Assimilation
- 20% chance to dodge an attack. Stackable with other items

BROADCAST
Requires 65% [link Defiled | | Defiled_Wastelands.html] Assimilation
- All enemies within a 20 tile radius of the player are inflicted with [link Rasterized]. Stackable only once with the [link Lousy Liver] or its variants

Bulwark
Requires 50% [link Riven | | Riven_Hive.html] Assimilation
- Gain a protective shell able to absorb all damage for a max of 200 damage before needing to regrow for 2 minutes

Threshold 4

Decay
Requires 90% [link Corruption | | https://terraria.wiki.gg/wiki/The_Corruption] Assimilation
- All enemies within a 50 tile radius of the player lose 2.1% of their max life each second

Bloodlust
Requires 86% [link Crimson | | https://terraria.wiki.gg/wiki/The_Crimson] Assimilation
- Replenish 4 health every second for each bleeding enemy within 40 tiles of you

INTERCONNECT
Requires 95% [link Defiled | | Defiled_Wastelands.html] Assimilation
- Defiled spikes will attack any enemies within 8 tiles from you. All attacks and destroyed projectiles prompt an [link Infusion explosion | | Infusion.html] dealing 1/5 of the weapon's
damage and benefit from any [link explosive | | Demolitionist_Weapons.html] boosts

Split Apart
Requires 75% [link Riven | | Riven_Hive.html] Assimilation
- Gain 50% Torn severity for 7 seconds after striking an enemy that goes on cooldown for 9 seconds after

Notes

- The player will still experience all of the negative effects of each assimilation type while this armor set is equipped.
- Only one ability of each threshold can be enabled at a time.

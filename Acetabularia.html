Acetabularia

[statblock Acetabularia]

Acetabularia is a plant grown on [link Riven Grass] or [link Riven Jungle Grass] [link blocks] found around the [link Riven Hive]. It is the Riven-equivalent to [link the Corruption's | | https://terraria.wiki.gg/wiki/The_Corruption] [link Vile Mushroom | | https://terraria.wiki.gg/wiki/Vile_Mushroom] and [link the Crimson's | | https://terraria.wiki.gg/wiki/The_Crimson] [link Vicious Mushroom | | https://terraria.wiki.gg/wiki/Vicious_Mushroom].

Missile Armcannon

[statblock Missile_Armcannon]

The Missile Armcannon is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] crafted using [link Destructive Claws] and a [link Gun Glove]. Swinging while this is equipped will fire any [link rockets | | https://terraria.wiki.gg/wiki/Rockets] in the player's inventory from their swinging-hand. Additionally, equipping the Armcannon provides these benefits:

- +30% [link thrown | | https://terraria.wiki.gg/wiki/Throwing_weapons] [link explosive | | Demolitionist_Weapons.html] [link velocity | | https://terraria.wiki.gg/wiki/Velocity]
- +10% explosive [link attack speed | | https://terraria.wiki.gg/wiki/Attack_speed]
- +15% explosive blast radius
- +50% explosive [link knockback | | knockback | | https://terraria.wiki.gg/wiki/Knockback]
- [link Auto-use | | https://terraria.wiki.gg/wiki/Autoswing] on all explosive weapons

Tips

- Like the [link Gun Glove], the Missile Armcannon benefits from [link attack speed | | https://terraria.wiki.gg/wiki/Attack_speed] boosts.
- The Missile Armcannon is effective with the [link Mine Flayer] as it is a swung explosive weapon. Combining their abilities with a [link Nitro Crate] or its upgrades
can quickly allow the player to clear their screen so long as they have enough ammunition and protection from their own explosives.

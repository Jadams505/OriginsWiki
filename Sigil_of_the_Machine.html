Sigil of the Machine

[statblock Sigil_of_the_Machine]

Sigil of the Machine is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link accessory | | Accessories.html] crafted using mech components dropped by the [link mechanical bosses | | https://terraria.wiki.gg/wiki/Mechanical_bosses]. Equipping it transforms the player into a mech that is able to use two weapons at once with the Use item and Interact buttons respectively. Alternate use items are not able to have their secondary abilities because of this. Furthermore, the player's defense is increased by 50% and [link mana capacity | | https://terraria.wiki.gg/wiki/Mana] doubled, are unaffected by [link fall damage | | https://terraria.wiki.gg/wiki/Fall_damage] but have their [link flight capabilities | | https://terraria.wiki.gg/wiki/Flight] halved, and are capable of dashing by double-tapping the horizontal movement keys. Like most transformations, the player will adopt a unique appearance while this is equipped.

Danger Barrel

[statblock Danger_Barrel]

The Danger Barrel is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] combining a [link Magma Stone | | https://terraria.wiki.gg/wiki/Magma_Stone] with an [link I-WTPA Standard | | IWTPA_Standard.html]. While equipped, all [link explosives | | Demolitionist_Weapons.html] inflict [link On Fire! | | https://terraria.wiki.gg/wiki/On_Fire!] and have a 33% reduced [link fuse time | | Fuse_Time.html].

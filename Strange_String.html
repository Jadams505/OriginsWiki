Lore

Strange String

Strange String is only part of a Defiled creature in which the curse cannot touch as it is the nervous system of the original host. The Defiled uses the nervous
system as a means to control each Defiled creature that roams the Wastelands. The Defiled spreads as much as it can to reach more potential hosts as it cannot
recover the nervous system of downed Defiled creatures.

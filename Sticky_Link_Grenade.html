Sticky Link Grenade

[statblock Link_Grenade]

The Sticky Link Grenade is an [link explosive | | Demolitionist_Weapons.html] that can be crafted using [link Gel | | https://terraria.wiki.gg/wiki/Gel] and [link Link Grenades]. It is functionally identical to standard Link Grenades, but now it can stick to tiles and enemies.

Tips

- Sticky Link Grenades can synergize well with projectile duplication bonuses. The player can quickly stick a lot of Sticky Link Grenades to one enemy and then attack them with another explosive to deal impressive damage in one hit.

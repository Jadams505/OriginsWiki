Crystal Grenade

[statblock Crystal_Grenade]

The Crystal Grenade is a type of [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link explosive | | Demolitionist_Weapons.html] that can be crafted using [link Grenades | | https://terraria.wiki.gg/wiki/Grenade] and [link Crystal Shards | | https://terraria.wiki.gg/wiki/Crystal_Shard]. Upon detonation, the Crystal Grenade releases 8-11 crystal shards that bounce and pierce enemies for a moment.

Crystal Grenade

[statblock Crystal_Grenade]

The Crystal Grenade is a type of Hardmode explosive that can be crafted using Grenades and Crystal Shards. Upon detonation, the Crystal Grenade releases 8-11 crystal shards that bounce and pierce enemies for a moment.

Crafting

Recipes

{ stations:[ 'Any Hardmode Anvil' ] items:[ { result:'Crystal Grenade(8)' ingredients:[ 'Crystal Shard' 'Grenade(8)' ] } ] }
Tips

- The Crystal Grenade excels when used in an enclosed space so the shards can affect more enemies during their lifetime.

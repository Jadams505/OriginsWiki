Fervor Potion

[statblock Fervor_Potion]

The Fervor Potion is a [link buff potion | | Buff_Potions.html] that can be crafted using [link Bottled Water | | https://terraria.wiki.gg/wiki/Bottled_Water], a [link Bilemouth] and any [link evil herb | | Recipes.html]. When consumed, it provides the Fervor [link buff | | Buffs.html], which increases weapon speed by 10%. This effect will last for 4 minutes, but can be canceled anytime by right-clicking the icon.

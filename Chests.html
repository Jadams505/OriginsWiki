Chests

If you are looking for the official page of vanilla chests, click [link here | | https://terraria.wiki.gg/wiki/Chests].

[statblock Chests]

Chests are [link storage items | | https://terraria.wiki.gg/wiki/Storage_items] that each hold up to 40 item stacks. Chests can be broken with any [link pickaxe | | Pickaxes.html] or [link drill | | Drills.html], but only when empty. It is impossible to destroy the blocks underneath a placed Chest that contains items.

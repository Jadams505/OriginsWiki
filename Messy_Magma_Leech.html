The Messy Magma Leech is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] combining a [link Magma Stone | | https://terraria.wiki.gg/wiki/Magma_Stone] and a [link Messy Leech]. While equipped, all attacks from the player inflict the [link Bleeding | | https://terraria.wiki.gg/wiki/Bleeding] and [link On Fire! | | https://terraria.wiki.gg/wiki/On_Fire!] debuffs.

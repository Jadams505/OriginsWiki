Switchblade

[statblock Switchblade]

The Switchblade is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link melee weapon | | Melee_Weapons.html] that is crafted using [link Sanguinite Bars | | Sanguinite_Bar.html]. The swords function can be switched by pressing the Open/Activate button, changing between a [link shortsword | | Shortswords.html] and [link broadsword | | Broadsword.html] function. It deals more damage in shortsword form, but has more range in its broadsword form.

Notes

- The Switchblade's [link attack speed | | https://terraria.wiki.gg/wiki/Attack_speed] does not change depending on the mode.

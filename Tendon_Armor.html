Tendon Armor

[statblock Tendon_Armor]

Tendon armor is an [link armor] set consisting of the Tendon Helmet, Tendon Shirt and Tendon Pants. In order to craft the full set the player will need 42 [link Crimtane Ore | | https://terraria.wiki.gg/wiki/Crimtane_Ore] and 66 [link Vertebrae | | https://terraria.wiki.gg/wiki/Vertebra].

A full set of Tendon armor provides the following benefits:

- +10 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +10% [link ranged | | Ranged_Weapons.html] damage
- +6% [link critical strike | | https://terraria.wiki.gg/wiki/Critical_hit] chance
- +20% ammo conservation
Set bonus
- Movement speed, acceleration/deceleration and jump speed are multiplied by the player's current life

Bile Dart

[statblock Bile_Dart]

The Bile Dart is a type of Hardmode dart fired by various dart launchers such as the Blowpipe, Blowgun, [link Dart Crossbow], Dart Pistol, Dart Rifle, [link Fungi Pipe] and [link Incision]. Bile darts generate an aura around them which inflict [link Rasterized] for one second to any enemies within it.

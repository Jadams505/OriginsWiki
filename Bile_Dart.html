Bile Dart

[statblock Bile_Dart]

The Bile Dart is a type of [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link dart | | Darts.html] fired by [link dart launchers]. Bile darts generate an aura around them which inflict [link Rasterized] for half a second for direct hits and 0.2 seconds for near misses. The aura also damages enemies during near misses as well.

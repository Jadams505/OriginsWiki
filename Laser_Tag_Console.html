Laser Tag Console

The Laser Tag Console is a functional furniture item used to initiate and customize the Laser Tag event. When the player presses the Open/Activate button on the console a menu will appear where the player can modify the players involved and time of the event, and can toggle between starting and ending it using the same key. Upon starting the event, any players with a Laser Tag Vest will be able to participate.

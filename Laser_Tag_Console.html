Laser Tag Console

[statblock Laser_Tag_Console]

The Laser Tag Console is a functional [link furniture] item used to initiate and customize the [link Laser Tag] [link event | | Events.html]. When the player presses the Open/Activate button on the console a menu will appear where the player can modify the players involved and time of the event, and can toggle between starting and ending it using the same key. Upon starting the event, any players with a [link Laser Tag Vest] will be able to participate.

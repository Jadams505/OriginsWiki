Abyssal Anchor

[statblock Abyssal_Anchor]

The Abyssal Anchor is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode], post-[link Golem | | https://terraria.wiki.gg/wiki/Golem] [link accessory | | Accessories.html] that is crafted with the [link Binding Book] and [link Celestial Stone Mask]. Equipping it provides major boosts to the player's stats at the cost of their movement speed.

Stat boosts:
- +15% melee speed
- +10% [link generic damage]
- +5% [link magic damage | | Magic_Weapons.html]
- +4% [link critical strike | | https://terraria.wiki.gg/wiki/Critical_hit] chance
- +4 HP/s [link health regeneration | | https://terraria.wiki.gg/wiki/Health_regeneration]
- +20 [link mana | | https://terraria.wiki.gg/wiki/Mana] capacity
- +8 defense
- +5% [link damage reduction | | https://terraria.wiki.gg/wiki/Damage_reduction]
- +25% [link mana shielding]
- +25% [link mining speed | | https://terraria.wiki.gg/wiki/Mining_speed]
- +1 minion [link knockback | | https://terraria.wiki.gg/wiki/Knockback]
- Greatly increased [link pickup range | | https://terraria.wiki.gg/wiki/Items#Pickup] for [link Stars | | https://terraria.wiki.gg/wiki/Star]

Stat reduction:
- -25% movement speed
- -40% jump speed

Notes

- Abyssal Anchor's effect is stackable with other mana shielding boosts (see [link item synergies | | Item_Synergies#Mana_Shielding.html]).
- Mana shielding takes into account the damage the player would have received with 0 defense and 0% damage reduction.
This results in the proper amount of damage reduced from the attack, but an inefficiency in the mana consumed from it. - Whenever the player benefits from mana shielding, [link Defiled Exhaustion] will be applied preventing any mana regneration for 3.2 seconds.
- The Binding Book does not stack with [link Magic Cuffs | | https://terraria.wiki.gg/wiki/Magic_Cuffs] or any of its variants for decreased mana consumption.

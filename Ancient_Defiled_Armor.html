Ancient Defiled Armor

[statblock Ancient_Defiled_Armor]

Ancient Defiled armor is an armor set that reproduces the pre-release design, and is otherwise identical in all ways to the current Defiled armor set. Each piece is interchangeable with the equivalent Defiled armor piece, while still granting the same set bonus.

Each piece has a 1/525 (0.19%) chance of being dropped by most Defiled enemies.

Comb

[statblock Comb]

The Comb is a rare [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] that is guaranteed to drop from [link The Groom | | https://terraria.wiki.gg/wiki/The_Groom]. Equipping it grants a 3% boost to all damage types. It is the offensive-equivalent to the [link Shackle | | https://terraria.wiki.gg/wiki/Shackle].

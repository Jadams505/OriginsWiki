Nova Fragment

[statblock Nova_Fragment]

The Nova Fragments are a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode], post-[link Lunatic Cultist | | https://terraria.wiki.gg/wiki/Lunatic_Cultist] [link material | | Materials.html] that is dropped from the [link Nova Pillar] during the [link Lunar Events | | https://terraria.wiki.gg/wiki/Lunar_Events] in stacks of 12-60 (24-100 in [link Expert Mode | | https://terraria.wiki.gg/wiki/Expert_Mode]). Nova Fragments are used to craft [link Nova armor], a late-game armor set that benefits all types of [link explosives | | Demolitionist_Weapons.html], as well as Nova-themed [link tools], [link weapons], [link wings], etc.

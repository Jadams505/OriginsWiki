Combat Core

[statblock Combat_Core]

The Combat Core is a post-[link Moon Lord | | https://terraria.wiki.gg/wiki/Moon_Lord], Eyndum-tier [link accessory | | Accessories.html] that is crafted from [link materials] obtained through the [link Test of the Ancients] [link event | | Events.html] at an [link Elderforge]. It can only be equipped in a dedicated core slot provided by a full set of [link Eyndum Armor] that appears to the left of the player's equipment section. It provides the following boosts:

- +24% [link generic damage] - +35% generic [link critical strike chance | | https://terraria.wiki.gg/wiki/Critical_hit] - +2 generic [link knockback | | https://terraria.wiki.gg/wiki/Knockback]

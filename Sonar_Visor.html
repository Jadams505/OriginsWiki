Sonar Visor

[statblock Sonar_Visor]

The Sonar Visor is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] post-[link Plantera | | https://terraria.wiki.gg/wiki/Plantera] [link accessory | | Accessories.html] crafted using an [link Ultrabright Helmet | | https://terraria.wiki.gg/wiki/Ultrabright_Helmet] and [link Advanced Imaging]. It provides greater visibility in the dark by highlighting tiles nearby the player. It grants a unique visual effect when worn.

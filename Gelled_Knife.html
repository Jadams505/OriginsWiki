Gelled Knife

[statblock Gelled_Knife]

The Gelled Knife is a [link consumable | | Consumables.html] [link ranged weapon | | Ranged_Weapons.html] that is crafted using [link Throwing Knives | | https://terraria.wiki.gg/wiki/Throwing_Knife] and [link Sentient Powder]. Each knife has a 1/2 (50%) chance to inflict [link Torn] that increases in severity depending on how many consecutive times the targeted enemy has been struck by a Gelled Knife. They otherwise behave exactly like the Throwing Knives used to create them, except for having slightly better statistics overall, along with [link auto-use | | https://terraria.wiki.gg/wiki/Autoswing].

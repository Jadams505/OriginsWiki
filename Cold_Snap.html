Cold Snap

[statblock Cold_Snap]

The Cold Snap is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link launcher | | Launchers.html] crafted using [link Ice Blocks | | https://terraria.wiki.gg/wiki/Ice_Blocks] and [link Shiverthorn | | https://terraria.wiki.gg/wiki/Shiverthorn]. The canisters launched from it explode into blue light and inflict [link Frostburn | | https://terraria.wiki.gg/wiki/Frostburn] to anything in the blast radius.

Its best [link modifier | | Modifiers.html] is Guileless.

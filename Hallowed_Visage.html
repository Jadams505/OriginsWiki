Hallowed Visage

[statblock Hallowed_Visage]

The Hallowed Visage is a helmet [link armor] piece to serve as the [link Demolitionist | | Demolitionist_Weapons.html]-equivalent for [link Hallowed armor | | https://terraria.wiki.gg/wiki/Hallowed_armor]. Equipping this as part of a full set does not change the set bonus provided and only offers basic combat stats.

The Hallowed Visage provides the following benefits:

- +3 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +15% explosive [link velocity | | https://terraria.wiki.gg/wiki/Velocity]
- +8% explosive [link critical strike | | https://terraria.wiki.gg/wiki/Critical_hit] chance

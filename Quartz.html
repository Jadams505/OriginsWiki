Quartz Block

[statblock Quartz]

Quartz Block is a [link block | | Blocks.html] type found in an [link Underground Desert | | https://terraria.wiki.gg/wiki/Underground_Desert] converted by the [link Riven | | Riven_Hive.html]. It is the Riven-equivalent to the [link hardened sand blocks | | https://terraria.wiki.gg/wiki/Hardened_Sand_Blocks].

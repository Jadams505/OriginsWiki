Boomerang Magnet

[statblock Boomerang_Magnet]

The Boomerang Magnet is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] combining the [link Treasure Magnet | | https://terraria.wiki.gg/wiki/Treasure_Magnet] and the [link Turbo Reel]. The Boomerang Magnet increases the return speed of [link boomerangs | | https://terraria.wiki.gg/wiki/Boomerangs] and [link harpoon guns | | Harpoon_Guns.html] while also increasing the [link pickup range | | https://terraria.wiki.gg/wiki/Items#Pickup] for items.

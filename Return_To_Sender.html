Return To Sender

{statblock { header:Return To Sender, items:[ {image: Images/Return_To_Sender.png} ] }, { header:Statistics, items:[ { label:Tooltip, values:[ "Contact damage is reciprocated to 40% of its original amount" ] }, { label:Rarity, value:[link | Images/RareBlue.png | https://terraria.wiki.gg/wiki/Rarity] }, { label:Sell, value:[value 0|0|60|0] }, { label:Research, value:1 required } ] } statblock}

Return To Sender is an [link accessory | $default | Accessories.html] dropped by the [link Defiled Amalgamation] for a 33% chance. When equipped, Return To Sender will apply 40% of the contact damage to the enemy that came in contact with the player (50% in Expert, and 60% in Master), but will not block any damage done to the player.

Notes

- Return To Sender will still deal damage to enemies upon the player's death making it possible to defeat a boss post-mortem.

- The effects of Return To Sender are stackable with 'Thorns'.

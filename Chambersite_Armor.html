Chambersite Armor

[statblock Chambersite_Armor]

Chambersite armor is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode], post-[link Plantera | | https://terraria.wiki.gg/wiki/Plantera] [link armor] set consisting of the Chambersite Helmet, Chambersite Breastplate and Chambersite Greaves. In order to craft the full set the player will need 9 [link Solar Tablet Fragments | | https://terraria.wiki.gg/wiki/Solar_Tablet_Fragment], 24 [link Chambersite] and 72 [link Carburite].

A full set of Chambersite armor provides the following benefits:

- +21 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +10% [link generic damage]
- +10% generic [link velocity | | https://terraria.wiki.gg/wiki/Velocity]
- +10% generic [link critical strike | | https://terraria.wiki.gg/wiki/Critical_hit] chance
- +8% [link damage reduction | | https://terraria.wiki.gg/wiki/Damage_reduction]
- +10% generic [link attack speed | | https://terraria.wiki.gg/wiki/Attack_speed]
- +10% movement speed
Set bonus
- Summons a chambersite sentinel to protect you
- Gain [link Voidsight]

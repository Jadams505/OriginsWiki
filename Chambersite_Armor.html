Chambersite Armor

Chambersite armor is a Hardmode, post-Plantera armor set consisting of the Chambersite Helmet, Chambersite Breastplate and Chambersite Greaves. In order to craft the full set the player will need 9 Solar Tablet Fragments, 24 Chambersite and 72 Carburite.

A full set of Chambersite armor provides the following benefits:

- +21 defense
- +10% generic damage
- +10% generic velocity
- +10% generic critical strike chance
- +8% damage reduction
- +10% generic attack speed
- +10% movement speed
Set bonus
- Summons a chambersite sentinel to protect you
- Gain Voidsight

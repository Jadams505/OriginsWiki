Edenwood Armor

Edenwood armor is a Hardmode, post-Golem armor set consisting of the Edenwood Helmet, Edenwood Breastplate and Edenwood Greaves. In order to craft the full set the player will need 400 Edenwood.

A full set of Edenwood armor provides the following benefits:

- +27 defense
- +10% explosive critical strike chance
- +20% explosive blast radius
- +15% movement speed
- +25% jump speed
Set bonus
- Immune time after taking self-damage increased

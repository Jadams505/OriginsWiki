Edenwood Armor

[statblock Edenwood_Armor]

Edenwood armor is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode], post-[link Golem | | https://terraria.wiki.gg/wiki/Golem] [link armor] set consisting of the Edenwood Helmet, Edenwood Breastplate and Edenwood Greaves. In order to craft the full set the player will need 400 [link Edenwood].

A full set of Edenwood armor provides the following benefits:

- +27 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +10% [link explosive | | Demolitionist_Weapons.html] [link critical strike | | https://terraria.wiki.gg/wiki/Critical_hit] chance
- +20% explosive blast radius
- +15% movement speed
- +25% jump speed
Set bonus
- Summons a Edenwood sentinel to protect you
- Gain [link Voidsight]

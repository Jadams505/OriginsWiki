Sootgrass

[statblock Sootgrass]

Sootgrass is a [link block | | Blocks.html] type found around the [link Ashen Factory]. It is the Ashen-equivalent to [link Grass | | https://terraria.wiki.gg/wiki/Grasses], and can be converted to other variants with [link any thrown powder | | https://terraria.wiki.gg/wiki/Thrown_Powder]. Unlike the other types of grass, no foliage will grow on Sootgrass nor is it placeable/spread able through its own seed type in which it lacks.

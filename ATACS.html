ATACS

[statblock ATACS]

ATACS is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode], post-[link Golem | | https://terraria.wiki.gg/wiki/Golem] [link accessory | | Accessories.html] combining the [link Eye of the Golem | | https://terraria.wiki.gg/wiki/Eye_of_the_Golem] with the [link Strange Computer]. Equipping it shows an indicator of a projectile's directory from the weapon the player is currently holding and grants a +10% critical strike chance. ATACS talks to the player similarly to [link Lucy the Axe | | https://terraria.wiki.gg/wiki/Lucy_the_Axe].

Magic Core

[statblock Magic_Core]

The Magic Core is a post-[link Moon Lord | | https://terraria.wiki.gg/wiki/Moon_Lord], Eyndum-tier [link accessory | | Accessories.html] that is crafted from [link materials] obtained through the [link Test of the Ancients] [link event | | Events.html] at an [link Elderforge]. It can only be equipped in a dedicated core slot provided by a full set of [link Eyndum Armor] that appears to the left of the player's equipment section. It provides the following boosts:

- -20% [link mana cost | | https://terraria.wiki.gg/wiki/Mana]
- Automatically use [link mana potions | | ] when low on mana

Dusk Key

[statblock Dusk_Key]

The Dusk Key is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link consumable | | Consumables.html] item that has a 1/2500 (0.04%) chance of being dropped by any enemy in [link the Dusk]. Despite being unusable until [link Plantera | | https://terraria.wiki.gg/wiki/Plantera] has been defeated, it can be dropped at any time during Hardmode. It is used to unlock the [link Dusk Dungeon Chest] by right-clicking it while holding this.

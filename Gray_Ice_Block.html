Gray Ice Block

[statblock Gray_Ice]

Gray Ice Block is a [link block | | Blocks.html] type found in a [link Snow biome | | https://terraria.wiki.gg/wiki/Snow_biome] converted by the [link Defiled | | Defiled_Wastelands.html]. It is the Defiled-equivalent to [link ice blocks | | https://terraria.wiki.gg/wiki/Ice_Blocks], and can be converted to other variants with [link any thrown powder | | https://terraria.wiki.gg/wiki/Thrown_Powder].

Notes

- The Gray Ice Block can be purified, but will quickly reclaim lost ground. This is a part of the [link Defiled Resurgence] mechanic where the Defiled cannot be cleansed until all [link Defiled Hearts | | Defiled_Heart.html] are removed from the world.

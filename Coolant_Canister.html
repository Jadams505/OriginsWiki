Coolant Canister

[statblock Coolant_Canister]

The Coolant Canister is a type of [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode] [link canister | | Canisters.html] used as [link ammunition] for [link launchers | | Launchers.html] and the [link Proximity Mine Launcher | | https://terraria.wiki.gg/wiki/Proximity_Mine_Launcher]. Upon landing on a tile or enemy, the Coolant Canister will explode for a blast radius of 13 tiles and inflict [link Frostbite | | https://terraria.wiki.gg/wiki/Frostbite] to any enemies within.

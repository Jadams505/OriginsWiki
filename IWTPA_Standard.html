I-WTPA Standard

[statblock IWTPA_Standard]

The I-WTPA Standard is a [link pre-Hardmode | | https://terraria.wiki.gg/wiki/Pre-Hardmode] [link accessory | | Accessories.html] that has a 1/4 (25%) chance of being dropped by [link Undead Miners | | https://terraria.wiki.gg/wiki/Undead_Miner] found [link Underground | | https://terraria.wiki.gg/wiki/Underground]. It is also sold by the [link Demolitionist | | https://terraria.wiki.gg/wiki/Demolitionist] [link NPC | | NPCs.html] for [price 5|0|0] after selling him 35 [link Peat Moss]. While equipped, all [link explosive | | Demolitionist_Weapons.html] [link fuse times | | Fuse_Time.html] will be reduced by 33%.

Soulhide Armor

[statblock Soulhide_Armor]

Soulhide armor is an [link armor] set consisting of the Soulhide Mask, Soulhide Coat and Soulhide Guards. In order to craft the full set the player will need 42 [link Demonite Ore | | https://terraria.wiki.gg/wiki/Demonite_Ore] and 66 [link Rotten Chunk | | https://terraria.wiki.gg/wiki/Rotten_Chunk].

A full set of Soulhide armor provides the following benefits:

- +10 [link defense | | https://terraria.wiki.gg/wiki/Defense]
- +10% [link melee weapon | | Melee_Weapons.html] size
- +12% melee [link attack speed | | https://terraria.wiki.gg/wiki/Attack_speed]
- +14% [link knockback | | https://terraria.wiki.gg/wiki/Knockback]
Set bonus
- Enemies within 10 tiles of the player are inflicted with [link Shadowflame | | https://terraria.wiki.gg/wiki/Shadowflame]
- Enemies within 15 tiles of the player are inflicted with [link Weak | | https://terraria.wiki.gg/wiki/Weak]

Locus Armor

[statblock Locus_Armor]

Locus armor is a post-[link Moon Lord | $default | https://terraria.wiki.gg/wiki/Moon_Lord] [link armor] set consisting of the Locus Cranium, Locus Shell and Locus Leggings. In order to craft the full set the player will need 48 [link Batholith] and 42 [link Formium Bars | $default | Formium_Bar.html].

A full set of Locus armor provides the following benefits:

- +64 [link defense | $default | https://terraria.wiki.gg/wiki/Defense]
- +80 [link mana | $default | https://terraria.wiki.gg/wiki/Mana] capacity
- +4 HP/s [link life regeneration | $default | https://terraria.wiki.gg/wiki/Health_regeneration]
Set bonus
- Grants the ability to manipulate mana for several abilities
- A UI selection element will appear for each of the abilities

Abilities

Dual Function
Consumes 63 mana when teleported.
- Projects the player's silohuette in a random open space on screen every half second. Whenever the player would've been attacked, this instead teleports the player on the silohuette, avoiding the attack.

Closed System
Consumes the player's mana until 1 mana remains.
- All but one mana is used to charge up the current held weapon's stats based on the mana used.

Entropy
Consumes 6 mana each second.
- Any enemy within 80 tiles of the player is inflicted with Ionizing Radiation.

Notes

- If the player reaches less than or equal to 0 mana because of any ability from Locus armor, they will receive [link Ionizing Radiation] until the ability is deactivated.

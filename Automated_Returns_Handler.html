Automated Returns Handler

[statblock Automated_Handler]

The Automated Returns Handler is a [link Hardmode | $default | https://terraria.wiki.gg/wiki/Hardmode] post-[link Plantera | $default | https://terraria.wiki.gg/wiki/Plantera] [link accessory | $default | Accessories.html] crafted using Master Ninja Gear and a [link Super Turbo Reel]. While equipped, the player has the ability to climb and slide down walls, dash by double-tapping the horizontal movement keys, a 10% chance to dodge any attack and an increased return speed for grapple hooks and [link harpoons | $default | Harpoon_Guns.html] by 2.

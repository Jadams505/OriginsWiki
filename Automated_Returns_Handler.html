Automated Returns Handler

[statblock Automated_Handler]

The Automated Returns Handler is a [link Hardmode | | https://terraria.wiki.gg/wiki/Hardmode], post-[link Plantera | | https://terraria.wiki.gg/wiki/Plantera] [link accessory | | Accessories.html] crafted using [link Master Ninja Gear | | https://terraria.wiki.gg/wiki/Master_Ninja_Gear] and a [link Super Turbo Reel]. While equipped, the player has the ability to climb and slide down walls, dash by double-tapping the horizontal movement keys, a 10% chance to dodge any attack and an increased return speed for [link hooks | | https://terraria.wiki.gg/wiki/Hooks] and [link harpoons | | Harpoon_Guns.html] by 2.
